# Refer to More Than One Sheet

Use the  **Array** function to identify a group of sheets. The following example selects three sheets in the active workbook.


```vb
Sub Several() 
 Worksheets(Array("Sheet1", "Sheet2", "Sheet4")).Select 
End Sub
```
