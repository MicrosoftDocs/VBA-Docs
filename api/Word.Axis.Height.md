# Axis.Height property (Word)

Returns the height, in [points](../language/glossary/vbe-glossary.md#point), of the object. Read-only  **Double**.


## Syntax

_expression_.**Height**

_expression_ A variable that represents an **[Axis](Word.Axis.md)** object.
