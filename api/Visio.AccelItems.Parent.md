# AccelItems.Parent Property (Visio)

Determines the parent of an object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ A variable that represents a [AccelItems](./Visio.AccelItems.md) object.


## Return value

AccelTable


## Remarks


 **Note**  

In general, an object's parent is the object that contains it.
