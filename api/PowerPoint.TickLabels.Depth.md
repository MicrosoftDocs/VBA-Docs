# TickLabels.Depth Property (PowerPoint)

Returns the number of levels of category tick labels. Read-only  **Long**.


## Syntax

 _expression_. `Depth`

 _expression_ A variable that represents a '[TickLabels](PowerPoint.TickLabels.md)' object.
