# Hyperlink.Row Property (Visio)

Returns the row index of a hyperlink. Read-only.


## Syntax

 _expression_. `Row`

 _expression_ A variable that represents a [Hyperlink](./Visio.Hyperlink.md) object.


## Return value

Integer
