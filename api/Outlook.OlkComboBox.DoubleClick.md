# OlkComboBox.DoubleClick Event (Outlook)

Occurs when the user double-clicks inside the control.


## Syntax

_expression_. `DoubleClick`

_expression_ A variable that represents an [OlkComboBox](./Outlook.OlkComboBox.md) object.
