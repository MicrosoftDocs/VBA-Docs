# ErrorBars.Format property (Excel)

Returns the **[ChartFormat](Excel.ChartFormat.md)** object. Read-only.


## Syntax

_expression_.**Format**

_expression_ A variable that represents an **[ErrorBars](excel.errorbars(object).md)** object.
