# OLEObject.BringToFront Method (Excel)

Brings the object to the front of the z-order.


## Syntax

 _expression_. `BringToFront`

 _expression_ A variable that represents an [OLEObject](Excel.OLEObject.md) object.


### Return Value

Variant
