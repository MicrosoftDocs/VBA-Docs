# CustomLayout.Select Method (PowerPoint)

Selects the specified  **[CustomLayout](PowerPoint.CustomLayout.md)** object.


## Syntax

 _expression_. `Select`

_expression_ A variable that represents a [CustomLayout](./PowerPoint.CustomLayout.md) object.
