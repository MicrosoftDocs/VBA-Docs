# ShapeRange.Chart property (Project)
Gets a  **Chart** object that represents the chart contained in the shape range. Read-only **Chart**.

## Syntax

_expression_.**Chart**

_expression_ A variable that represents a 'ShapeRange' object.


## Property value

 **CHART**
