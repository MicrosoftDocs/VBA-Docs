# TextRange2.Paste Method (PowerPoint)

Pastes the contents of the Clipboard into the  **TextRange2** object.


## Syntax

 _expression_. `Paste`

 _expression_ An expression that returns a 'TextRange2' object.


### Return value

TextRange2
