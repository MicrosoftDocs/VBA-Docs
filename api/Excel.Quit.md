# Quit Method

Quits Microsoft Graph.

_expression_. `Quit`

 _expression_ Required. An expression that returns an [Application](Excel.Application-graph-property.md) object.


## Example

This example quits Microsoft Graph.


```vb
myChart.Application.Quit
```
