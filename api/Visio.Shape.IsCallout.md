# Shape.IsCallout property (Visio)

Indicates whether the shape is a callout shape. Read-only.


## Syntax

_expression_. `IsCallout`

_expression_ A variable that represents a **[Shape](Visio.Shape.md)** object.


## Return value

 **Boolean**
