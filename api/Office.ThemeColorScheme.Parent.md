# ThemeColorScheme.Parent property (Office)

Gets the **Parent** object for the **ThemeColorScheme** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ An expression that returns a **[ThemeColorScheme](Office.ThemeColorScheme.md)** object.


## Return value

Object
