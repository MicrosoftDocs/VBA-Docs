# Application.EnableChangeHighlighting property (Project)

 **True** if change highlighting is enabled. Read/write **Boolean**.


## Syntax

_expression_. `EnableChangeHighlighting`

_expression_ A variable that represents an **[Application](Project.Application.md)** object.


## Remarks

The default value of  **EnableChangeHighlighting** is **False**. See also the **[ToggleChangeHighlighting](Project.Application.ToggleChangeHighlighting.md)** method.
