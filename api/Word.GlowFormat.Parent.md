# GlowFormat.Parent property (Word)

Returns an **Object** that represents the parent object of the specified **GlowFormat** object.


## Syntax

_expression_.**Parent**

 _expression_ An expression that returns a [GlowFormat](./Word.GlowFormat.md) object.
