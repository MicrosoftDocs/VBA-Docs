# Chart.HasTitle property (Project)
 **True** if the axis or chart has a visible title. Read/write **Boolean**.

## Syntax

_expression_.**HasTitle**

_expression_ A variable that represents a **[Chart](Project.Chart.md)** object.


## Property value

 **BOOL**
