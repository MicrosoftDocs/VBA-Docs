# NewItemAlertRuleAction.ActionType Property (Outlook)

Returns a constant,  **olRuleActionNewItemAlert** , from the **[OlRuleActionType](Outlook.OlRuleActionType.md)** enumeration that indicates the type of action that is taken by the rule. Read-only.


## Syntax

 _expression_. `ActionType`

 _expression_ A variable that represents a [NewItemAlertRuleAction](./Outlook.NewItemAlertRuleAction.md) object.


## Remarks

 The new item alert rule action is supported only by rules for receiving messages. For more information, see [Specify Rule Actions](../outlook/How-to/Rules/specifying-rule-actions.md).
