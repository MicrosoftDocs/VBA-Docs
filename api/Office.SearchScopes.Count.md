# SearchScopes.Count property (Office)

Gets a  **Long** indicating the number of items in the **SearchScopes** object. Read-only.


## Syntax

_expression_. `Count`

_expression_ A variable that represents a [SearchScopes](Office.SearchScopes.md) object.


## Return value

Long
