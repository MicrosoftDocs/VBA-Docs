# PlaceholderFormat.Name Property (PowerPoint)

Returns or sets the name of the specified object. Read/write.


## Syntax

 _expression_. `Name`

 _expression_ A variable that represents a [PlaceholderFormat](./PowerPoint.PlaceholderFormat.md) object.


### Return value

String
