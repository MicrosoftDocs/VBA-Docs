# Series.AxisGroup property (Word)

Returns the type of axis group. Read/write  **[XlAxisGroup](Word.xlaxisgroup.md)**.


## Syntax

_expression_.**AxisGroup**

_expression_ A variable that represents a '[Series](Word.Series.md)' object.
