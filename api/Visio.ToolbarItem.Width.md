# ToolbarItem.Width Property (Visio)

Gets the width of an object in pixels. Read/write.


## Syntax

 _expression_. `Width`

 _expression_ A variable that represents a [ToolbarItem](./Visio.ToolbarItem.md) object.


## Return value

Integer
