# PolicyItem.Parent property (Office)

Gets the **Parent** object for the **PolicyItem** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ An expression that returns a **[PolicyItem](Office.PolicyItem.md)** object.


## Return value

Object
