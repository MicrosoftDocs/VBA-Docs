# xlWindowView enumeration (Excel)

Specifies the view showing in the window.



|Name|Value|Description|
|:-----|:-----|:-----|
| **xlNormalView**|1|Normal.|
| **xlPageBreakPreview**|2|Page break preview.|
| **xlPageLayoutView**|3|Page layout view.|
