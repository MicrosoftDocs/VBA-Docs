# PbPlacementType Enumeration (Publisher)

Represents the placement of envelopes in the printer tray.



|Name|Value|Description|
|:-----|:-----|:-----|
| **pbPlacementCenter**|3|Center|
| **pbPlacementLeft**|1|Left|
| **pbPlacementRight**|2|Right|
