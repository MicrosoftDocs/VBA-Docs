# PropertyPageSite.Parent property (Outlook)

Returns the parent  **Object** of the specified object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a [PropertyPageSite](Outlook.PropertyPageSite.md) object.
