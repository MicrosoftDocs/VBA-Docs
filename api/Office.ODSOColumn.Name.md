# ODSOColumn.Name property (Office)

Gets the name of a data field in a mail merge data source. Read-only.


## Syntax

_expression_.**Name**

_expression_ A variable that represents an **[ODSOColumn](Office.ODSOColumn.md)** object.


## Return value

String
