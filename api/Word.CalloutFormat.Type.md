# CalloutFormat.Type property (Word)

Returns or sets the callout type. Read/write  **MsoCalloutType**.


## Syntax

 _expression_. `Type`

 _expression_ Required. A variable that represents a '[CalloutFormat](Word.CalloutFormat.md)' object.
