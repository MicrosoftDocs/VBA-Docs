# CustomXMLNode.FirstChild property (Office)

Gets a **CustomXMLNode** object corresponding to the first child element of the current node. If the node has no child elements (or if it isn't of type **[msoCustomXMLNodeElement](office.msocustomxmlnodetype.md)**), returns **Nothing**. Read-only.


## Syntax

_expression_.**FirstChild**

_expression_ An expression that returns a **[CustomXMLNode](Office.CustomXMLNode.md)** object.


## Return value

CustomXMLNode
