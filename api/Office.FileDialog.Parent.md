# FileDialog.Parent Property (Office)

Gets the  **Parent** object for the **FileDialog** object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ A variable that represents a [FileDialog](./Office.FileDialog.md) object.


### Return value

Object
