# SensitivityLabelInitInfo.UserId property (Office)

Use the **UserId** property to set the string representing the **[UPN](/azure/active-directory/hybrid/plan-connect-userprincipalname#what-is-userprincipalname)** of the identity of an Office Account signed into Office.

## Syntax

_expression_.**UserId**

_expression_ An expression that returns a **[SensitivityLabelInitInfo](Office.SensitivityLabelInitInfo.md)** object.


## Return value

String
