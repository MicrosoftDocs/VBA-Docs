# Slicer.Width Property (Excel)

Returns or sets the width of the specified slicer, in points. Read/write.


## Syntax

 _expression_. `Width`

 _expression_ A variable that represents a '[Slicer](Excel.Slicer.md)' object.


### Return value

Double
