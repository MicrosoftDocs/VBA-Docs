# Shape.Rotate90 Method (Visio)

Rotates an object 90 degrees counterclockwise.


## Syntax

 _expression_. `Rotate90`

 _expression_ A variable that represents a [Shape](./Visio.Shape.md) object.


## Return value

Nothing
