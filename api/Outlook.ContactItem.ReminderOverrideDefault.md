# ContactItem.ReminderOverrideDefault property (Outlook)

Returns or sets a **Boolean** value that is **True** if the reminder overrides the default reminder behavior for the item. Read/write.


## Syntax

_expression_. `ReminderOverrideDefault`

_expression_ A variable that represents a [ContactItem](Outlook.ContactItem.md) object.


## Remarks

You must set the  **ReminderOverrideDefault** property to validate the **[ReminderPlaySound](Outlook.ContactItem.ReminderPlaySound.md)** and the **[ReminderSoundFile](Outlook.ContactItem.ReminderSoundFile.md)** properties.
