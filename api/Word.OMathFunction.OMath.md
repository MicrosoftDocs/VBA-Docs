# OMathFunction.OMath property (Word)

Returns an  **[OMath](Word.OMath.md)** object that represents the equation. Read-only.


## Syntax

_expression_. `OMath`

 _expression_ An expression that returns an '[OMathFunction](Word.OMathFunction.md)' object.
