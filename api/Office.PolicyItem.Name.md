# PolicyItem.Name Property (Office)

Gets the name of the  **PolicyItem** object. Read-only.


## Syntax

 _expression_. `Name`

 _expression_ An expression that returns a [PolicyItem](./Office.PolicyItem.md) object.


## Return value

String
