# ChartSeries.ShowWaterfallConnectorLines property (Access)

**True** if connector lines are displayed for a Waterfall Modern Chart (only available in Current Channel). Read/write **Boolean**.


## Syntax

_expression_.**ShowWaterfallConnectorLines**

_expression_ A variable that represents a **[ChartSeries](Access.ChartSeries.md)** object.
