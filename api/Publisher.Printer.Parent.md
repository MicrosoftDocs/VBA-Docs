# Printer.Parent Property (Publisher)

Returns the parent of the  **[Printer](Publisher.Printer.md)** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a  **Printer** object.


## Return value

Object
