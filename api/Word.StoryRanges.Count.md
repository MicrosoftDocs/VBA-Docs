# StoryRanges.Count property (Word)

Returns a  **Long** that represents the number of story ranges in the collection. Read-only.


## Syntax

 _expression_. `Count`

 _expression_ Required. A variable that represents a '[StoryRanges](Word.storyranges.md)' collection.
