# ErrorBars.Border property (PowerPoint)

Returns the border of the object. Read-only  **[ChartBorder](PowerPoint.ChartBorder.md)**.


## Syntax

_expression_ A variable that represents an '[ErrorBars](PowerPoint.ErrorBars.md)' object.
