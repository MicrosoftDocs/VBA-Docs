# RulerLevels2.Parent property (Office)

Gets the **Parent** object for the **RulerLevels2** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ An expression that returns a **[RulerLevels2](Office.RulerLevels2.md)** object.


## Return value

Object
