# DialogSheetView.Sheet property (Excel)

Returns the sheet name for the specified **DialogSheetView** object. Read-only.


## Syntax

_expression_.**Sheet**

_expression_ A variable that represents a **[DialogSheetView](Excel.DialogSheetView.md)** object.
