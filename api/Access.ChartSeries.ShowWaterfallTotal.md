# ChartSeries.ShowWaterfallTotal property (Access)

**True** if the total is displayed for a Waterfall Modern Chart (only available in Current Channel). Read/write **Boolean**.


## Syntax

_expression_.**ShowWaterfallTotal**

_expression_ A variable that represents a **[ChartSeries](Access.ChartSeries.md)** object.
