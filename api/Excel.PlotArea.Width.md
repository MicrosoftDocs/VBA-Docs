# PlotArea.Width Property (Excel)

Returns or sets a  **Double** value that represents the width, in points, of the object.


## Syntax

 _expression_. `Width`

 _expression_ A variable that represents a [PlotArea](Excel.PlotArea(Graph property).md) object.
