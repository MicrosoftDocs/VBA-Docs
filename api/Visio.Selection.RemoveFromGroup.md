# Selection.RemoveFromGroup Method (Visio)

Removes selected shapes from a group.


## Syntax

 _expression_. `RemoveFromGroup`

 _expression_ A variable that represents a [Selection](./Visio.Selection.md) object.


## Return value

Nothing
