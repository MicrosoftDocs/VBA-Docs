# WdMultipleWordConversionsMode Enumeration (Word)

Specifies the direction for conversion between Hangul and Hanja.



|Name|Value|Description|
|:-----|:-----|:-----|
| **wdHangulToHanja**|0|Hangul to Hanja.|
| **wdHanjaToHangul**|1|Hanja to Hangul.|
