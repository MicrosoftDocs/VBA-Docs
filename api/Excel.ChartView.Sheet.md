# ChartView.Sheet Property (Excel)

Returns the sheet name for the specified  **[ChartView](Excel.ChartView.md)** object. Read-only.


## Syntax

 _expression_. `Sheet`

 _expression_ A variable that represents a [ChartView](Excel.ChartView.md) object.
