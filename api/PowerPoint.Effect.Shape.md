# Effect.Shape property (PowerPoint)

Returns a **[Shape](PowerPoint.Shape.md)** object that represents an animated shape.


## Syntax

_expression_.**Shape**

_expression_ A variable that represents an [Effect](PowerPoint.Effect.md) object.


## Return value

Shape
