# AcBrowseToObjectType enumeration (Access)

Specifies the type of object to browse to.

<br/>

|Name|Value|Description|
|:-----|:-----|:-----|
|**acBrowseToForm**|2|Open a form.|
|**acBrowseToReport**|3|Open a report.|
