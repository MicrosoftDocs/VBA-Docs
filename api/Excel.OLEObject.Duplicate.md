# OLEObject.Duplicate Method (Excel)

Duplicates the object and returns a reference to the new copy.


## Syntax

 _expression_. `Duplicate`

 _expression_ A variable that represents an [OLEObject](Excel.OLEObject.md) object.


### Return value

Object
