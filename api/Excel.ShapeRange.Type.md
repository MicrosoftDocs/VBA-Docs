# ShapeRange.Type property (Excel)

Returns an **[MsoShapeType](Office.MsoShapeType.md)** value that represents the shape type.


## Syntax

_expression_.**Type**

_expression_ A variable that represents a **[ShapeRange](Excel.shaperange.md)** object.
