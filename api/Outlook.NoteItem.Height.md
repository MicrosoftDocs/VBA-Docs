# NoteItem.Height property (Outlook)

Returns or sets a **Long** specifying the height (in pixels) of the note window. Read/write.


## Syntax

_expression_.**Height**

_expression_ A variable that represents a [NoteItem](Outlook.NoteItem.md) object.
