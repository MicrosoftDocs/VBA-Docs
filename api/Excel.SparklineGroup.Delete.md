# SparklineGroup.Delete Method (Excel)

Deletes the sparkline group.


## Syntax

 _expression_. `Delete`

 _expression_ A variable that represents a '[SparklineGroup](Excel.SparklineGroup.md)' object.


### Return value

Nothing
