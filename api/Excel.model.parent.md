# Model.Parent property (Excel)

Returns an **Object** that represents the parent object of the specified object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a **[Model](Excel.Model.md)** object.


## Property value

**OBJECT**
