# Endnotes.Count property (Word)

Returns a  **Long** that represents the number of endnotes in the collection. Read-only.


## Syntax

 _expression_. `Count`

 _expression_ Required. A variable that represents an '[Endnotes](Word.endnotes.md)' collection.
