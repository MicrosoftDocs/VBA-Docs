# CustomLayouts.Parent property (PowerPoint)

Returns the parent object for the specified object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a [CustomLayouts](PowerPoint.CustomLayouts.md) object.


## Return value

Object
