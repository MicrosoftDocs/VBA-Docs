# MetaProperty.Value Property (Office)

Gets or sets the value of the  **MetaProperty** object. Read/write.


## Syntax

 _expression_. `Value`

 _expression_ An expression that returns a [MetaProperty](./Office.MetaProperty.md) object.


### Return value

Variant
