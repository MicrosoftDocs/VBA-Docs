# SlicerItem.Parent Property (Excel)

Returns the parent  **[SlicerCache](Excel.SlicerCache.md)** object for the slicer item. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ A variable that represents a '[SlicerItem](Excel.SlicerItem.md)' object.


### Return value

SlicerCache
