# Point.Shadow property (PowerPoint)

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean**.


## Syntax

_expression_.**Shadow**

_expression_ A variable that represents a '[Point](PowerPoint.Point.md)' object.
