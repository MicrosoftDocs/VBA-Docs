# Comment.AssociatedObject Property (Visio)

Returns the object associated with the specified comment. Read-only  **Object**.


## Syntax

 _expression_. `AssociatedObject`

 _expression_ A variable that represents a 'Comment' object.


## Property value

 **OBJECT**
