# AutoFormatRule.Parent property (Outlook)

Returns the parent  **Object** of the specified object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents an [AutoFormatRule](Outlook.AutoFormatRule.md) object.
