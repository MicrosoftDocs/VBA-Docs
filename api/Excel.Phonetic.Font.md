# Phonetic.Font property (Excel)

Returns a  **[Font](Excel.Font(object).md)** object that represents the font of the specified object.


## Syntax

_expression_. `Font`

_expression_ A variable that represents a [Phonetic](Excel.Phonetic.md) object.
