# AnimationBehavior.Delete method (PowerPoint)

Deletes the specified  **AnimationBehavior** object.


## Syntax

_expression_.**Delete**

_expression_ A variable that represents an [AnimationBehavior](PowerPoint.AnimationBehavior.md) object.
