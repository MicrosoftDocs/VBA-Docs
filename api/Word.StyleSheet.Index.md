# StyleSheet.Index property (Word)

Returns a  **Long** that represents the position of an item in a collection. Read-only.


## Syntax

_expression_.**Index**

_expression_ Required. A variable that represents a '[StyleSheet](Word.StyleSheet.md)' object.
