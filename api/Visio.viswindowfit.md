# VisWindowFit Enumeration (Visio)

Values returned by the  **Window.ViewFit** property.



|Constant|Value|Description|
|:-----|:-----|:-----|
| **visFitNone**|0|No auto-fit.|
| **visFitPage**|1|Fit whole page.|
| **visFitWidth**|2|Fit to page width.|
