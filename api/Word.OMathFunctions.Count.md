# OMathFunctions.Count property (Word)

Returns the number of items in the  **OMathFunctions** collection. Read-only **Long**.


## Syntax

_expression_.**Count**

_expression_ A variable that represents a '[OMathFunctions](Word.OMathFunctions.md)' object.
