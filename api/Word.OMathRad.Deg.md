# OMathRad.Deg property (Word)

Returns an  **[OMath](Word.OMath.md)** object that represents the degree for a radical. Read-only.


## Syntax

_expression_. `Deg`

 _expression_ An expression that returns an '[OMathRad](Word.OMathRad.md)' object.
