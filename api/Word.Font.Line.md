# Font.Line property (Word)

Returns a [LineFormat](Word.LineFormat.md) object that specifies the formatting for a line. Read/write.


## Syntax

 _expression_. `Line`

 _expression_ An expression that returns a '[Font](Word.Font.md)' object.
