# OLEObjects.SendToBack method (Excel)

Sends the object to the back of the z-order.


## Syntax

_expression_.**SendToBack**

_expression_ A variable that represents an **[OLEObjects](Excel.OLEObjects.md)** object.


## Return value

Variant
