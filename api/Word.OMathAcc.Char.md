# OMathAcc.Char property (Word)

Returns or sets an **Integer** that represents the accent character for the accent object. Read/write.


## Syntax

_expression_. `Char`

 _expression_ An expression that returns an '[OMathAcc](Word.OMathAcc.md)' object.
