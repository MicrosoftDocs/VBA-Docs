# COMAddIns.Count property (Office)

Gets a count of the number of COM add-ins in the host application. Read-only.


## Syntax

_expression_.**Count**

_expression_ An expression that returns a **[COMAddIns](Office.COMAddIns.md)** object.


## Return value

Long
