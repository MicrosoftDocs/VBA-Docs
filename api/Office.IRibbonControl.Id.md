# IRibbonControl.Id property (Office)

Gets the ID of the control specified in the Ribbon XML markup customization file. Read-only.


## Syntax

_expression_.**Id**

_expression_ An expression that returns an **[IRibbonControl](Office.IRibbonControl.md)** object.


## Return value

String
