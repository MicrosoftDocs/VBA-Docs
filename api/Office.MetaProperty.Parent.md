# MetaProperty.Parent property (Office)

Gets the  **Parent** object for the **MetaProperty** object. Read-only.


## Syntax

_expression_. `Parent`

 _expression_ An expression that returns a [MetaProperty](Office.MetaProperty.md) object.


## Return value

Object
