# PeopleView.Filter property (Outlook)
Returns or sets a **String** value that represents the filter for a view. Read/write.

## Version information

Version Added: Outlook 2013 


## Syntax

_expression_. `Filter`

_expression_ A variable that represents a 'PeopleView' object.


## Property value

 **STRING**
