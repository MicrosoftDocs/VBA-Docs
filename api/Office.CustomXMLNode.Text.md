# CustomXMLNode.Text property (Office)

Gets or sets the text for the current node. Read/write.


## Syntax

 _expression_. `Text`

 _expression_ An expression that returns a [CustomXMLNode](./Office.CustomXMLNode.md) object.


## Return value

String
