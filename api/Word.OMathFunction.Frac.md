# OMathFunction.Frac property (Word)

Returns an  **[OMathFrac](Word.OMathFrac.md)** object that represents a fraction. Read-only.


## Syntax

_expression_. `Frac`

 _expression_ An expression that returns an '[OMathFunction](Word.OMathFunction.md)' object.
