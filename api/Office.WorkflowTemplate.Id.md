# WorkflowTemplate.Id Property (Office)

Gets the ID of a template used to create a workflow instance. Read-only.


## Syntax

 _expression_. `Id`

 _expression_ An expression that returns a [WorkflowTemplate](./Office.WorkflowTemplate.md) object.


### Return value

String
