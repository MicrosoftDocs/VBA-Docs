# XlBorderWeight enumeration (Excel)

Specifies the weight of the border around a range.



|Name|Value|Description|
|:-----|:-----|:-----|
| **xlHairline**|1|Hairline (thinnest border).|
| **xlMedium**|-4138|Medium.|
| **xlThick**|4|Thick (widest border).|
| **xlThin**|2|Thin.|
