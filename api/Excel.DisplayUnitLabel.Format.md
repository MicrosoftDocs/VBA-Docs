# DisplayUnitLabel.Format property (Excel)

Returns the **[ChartFormat](Excel.ChartFormat.md)** object. Read-only.


## Syntax

_expression_.**Format**

_expression_ A variable that represents a **[DisplayUnitLabel](excel.displayunitlabel(object).md)** object.
