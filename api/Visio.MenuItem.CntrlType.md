# MenuItem.CntrlType Property (Visio)

Gets or sets the control type of a menu item. Read/write.


## Syntax

 _expression_. `CntrlType`

 _expression_ A variable that represents a [MenuItem](./Visio.MenuItem.md) object.


## Return value

Integer
