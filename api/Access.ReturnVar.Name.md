# ReturnVar.Name property (Access)

Gets the name of the specified  **[ReturnVar](Access.ReturnVar.md)**. Read-only **String**.


## Syntax

_expression_. `Name`

_expression_ A variable that represents a [ReturnVar](Access.ReturnVar.md) object.
