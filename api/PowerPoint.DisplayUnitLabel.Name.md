# DisplayUnitLabel.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. `Name`

_expression_ A variable that represents a '[DisplayUnitLabel](PowerPoint.DisplayUnitLabel.md)' object.
