# XlLinkInfoType Enumeration (Excel)

Specifies the type of link.



|Name|Value|Description|
|:-----|:-----|:-----|
| **xlLinkInfoOLELinks**|2|OLE or DDE server|
| **xlLinkInfoPublishers**|5|Publisher|
| **xlLinkInfoSubscribers**|6|Subscriber|
