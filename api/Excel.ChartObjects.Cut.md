# ChartObjects.Cut method (Excel)

Cuts the object to the Clipboard.


## Syntax

_expression_. `Cut`

_expression_ A variable that represents a [ChartObjects](Excel.ChartObjects.md) object.


## Return value

Variant


## Remarks

Only embedded charts can be cut.
