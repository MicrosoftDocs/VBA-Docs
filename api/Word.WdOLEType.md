# WdOLEType enumeration (Word)

Specifies the type of an OLE object.



|Name|Value|Description|
|:-----|:-----|:-----|
| **wdOLEControl**|2|OLE control.|
| **wdOLEEmbed**|1|Embedded OLE object.|
| **wdOLELink**|0|Linked OLE object.|
