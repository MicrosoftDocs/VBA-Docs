# ShapeRange.Width Property (Project)
Gets or sets the width, in points, of the shapes within the range. Read/write  **Long**.

## Syntax

 _expression_. `Width`

 _expression_ A variable that represents a 'ShapeRange' object.


## Property value

 **FLOAT**
