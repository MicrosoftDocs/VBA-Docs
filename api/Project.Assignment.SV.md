# Assignment.SV Property (Project)

Gets the earned value scheduled variance (SV) of the assignment. Read-only  **Variant**.


## Syntax

 _expression_. `SV`

 _expression_ A variable that represents an [Assignment](./Project.Assignment.md) object.
