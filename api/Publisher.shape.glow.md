# Shape.Glow Property (Publisher)

Returns a  [GlowFormat](Publisher.glowformat.md) object that represents the glow formatting for a shape. Read-only.


## Syntax

 _expression_. **Glow**

 _expression_ A variable that represents a  **Shape** object.


## Return value

 **GLOWFORMAT**
