# Font2.Embedded Property (Office)

Gets a value specifying whether the font is embedded in a page. Read-only.


## Syntax

 _expression_. `Embedded`

 _expression_ An expression that returns a [Font2](./Office.Font2.md) object.


### Return Value

MsoTriState
