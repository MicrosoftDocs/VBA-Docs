# Calendar.ResourceGuid Property (Project)

Gets the GUID of the resource calendar. Read-only  **String**.


## Syntax

 _expression_. `ResourceGuid`

 _expression_ An expression that returns a [Calendar](./Project.Calendar.md) object.
