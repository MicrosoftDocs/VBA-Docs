# Day.Index property (Project)

Gets the index of a  **Day** object in the containing **Days** collection. Read-only **Integer**.


## Syntax

_expression_.**Index**

_expression_ A variable that represents a [Day](./Project.Day.md) object.


## Remarks

The  **Index** properties of other objects are used in similar ways. For an example, see the **[Index](Project.Project.Index.md)** property of the **Project** object.
