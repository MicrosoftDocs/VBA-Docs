# MsoBarType enumeration (Office)

Specifies the type of the command bar.



|Name|Value|Description|
|:-----|:-----|:-----|
|**msoBarTypeMenuBar**|1|Menu bar.|
|**msoBarTypeNormal**|0|Default command bar.|
|**msoBarTypePopup**|2|Shortcut menu.|
