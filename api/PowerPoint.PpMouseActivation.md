# PpMouseActivation enumeration (PowerPoint)

The action setting for a **MouseClick** or **MouseOver** event.



|Name|Value|Description|
|:-----|:-----|:-----|
|**ppMouseClick**|1|Mouse click|
|**ppMouseOver**|2|Mouse over|
