# Dialogs.Count property (Word)

Returns a  **Long** that represents the number of dialog boxes in the collection. Read-only.


## Syntax

 _expression_. `Count`

 _expression_ Required. A variable that represents a '[Dialogs](Word.dialogs.md)' collection.
