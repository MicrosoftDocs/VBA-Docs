# LegendKey.Delete method (Word)

Deletes the object.


## Syntax

_expression_.**Delete**

_expression_ A variable that represents a '[LegendKey](Word.LegendKey.md)' object.


## Remarks

Deleting a  **LegendKey** object deletes the entire series.
