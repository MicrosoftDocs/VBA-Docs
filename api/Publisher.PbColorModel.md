# PbColorModel Enumeration (Publisher)

Represents the color model of the picture.

|:-----|:-----|:-----|
| **pbColorModelCMYK**|2|CMYK|
| **pbColorModelGreyScale**|3|GreyScale|
| **pbColorModelRGB**|1|RGB|
| **pbColorModelUnknown**|4|Unknown|
