# ODSOColumns.Count property (Office)

Gets a  **Long** indicating the number of items in the **ODSOColumns** collection. Read-only.


## Syntax

_expression_. `Count`

_expression_ A variable that represents an [ODSOColumns](Office.ODSOColumns.md) object.


## Return value

Long
