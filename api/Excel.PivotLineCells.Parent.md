# PivotLineCells.Parent property (Excel)

Returns the parent object for the specified  **PivotLineCells** object. Read-only.


## Syntax

_expression_. `Parent`

_expression_ A variable that represents a [PivotLineCells](Excel.PivotLineCells.md) object.
