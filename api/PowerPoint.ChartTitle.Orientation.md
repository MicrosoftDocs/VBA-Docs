# ChartTitle.Orientation property (PowerPoint)

Returns or sets the text orientation. Read/write  **Long**.


## Syntax

_expression_.**Orientation**

_expression_ A variable that represents a '[ChartTitle](PowerPoint.ChartTitle.md)' object.


## Remarks

The value of this property can be an integer from -90 through 90 degrees or one of the  **[XlOrientation](PowerPoint.XlOrientation.md)** constants.
