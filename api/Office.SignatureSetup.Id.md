# SignatureSetup.Id property (Office)

Gets the ID of the signature provider for a document. Read-only.


## Syntax

_expression_.**Id**

_expression_ An expression that returns a **[SignatureSetup](Office.SignatureSetup.md)** object.


## Return value

String
