# TableObject.Delete Method (Excel)

Deletes the  **TableObject** object.


## Syntax

 _expression_. `Delete`

 _expression_ A variable that represents a [ModelTable Object (Excel)](Excel.modeltable.md) object.


### Return value

 **VOID**
