# PivotAxis.Parent Property (Excel)

Returns the parent object for the specified  **PivotAxis** object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ A variable that represents a [PivotAxis](Excel.PivotAxis.md) object.
