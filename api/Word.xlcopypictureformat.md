# XlCopyPictureFormat Enumeration (Word)

Specifies the format of the picture being copied.



|Name|Value|Description|
|:-----|:-----|:-----|
| **xlBitmap**|2|A bitmap (.bmp, .jpg, .gif).|
| **xlPicture**|-4147|A drawn picture (.png, .wmf, .mix).|
