# SharedWorkspaceFile.Parent property (Office)

Gets the  **Parent** object for the **SharedWorkspaceFile** object. Read-only.

> [!NOTE] 
> Beginning with Microsoft Office 2010, this object or member has been deprecated and should not be used.


## Syntax

_expression_. `Parent`

_expression_ A variable that represents a [SharedWorkspaceFile](Office.SharedWorkspaceFile.md) object.


## Return value

Object
