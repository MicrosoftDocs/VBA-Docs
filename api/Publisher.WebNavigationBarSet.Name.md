# WebNavigationBarSet.Name Property (Publisher)

Indicates the name of the specified Web navigation bar set. Read/write.


## Syntax

 _expression_. **Name**

 _expression_An expression that returns a  **WebNavigationBarSet** object.


## Return value

String
