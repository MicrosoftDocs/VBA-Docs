# TableStyle.BuiltIn property (Excel)

**True** if the style is a built-in style. Read-only **Boolean**.


## Syntax

_expression_.**BuiltIn**

_expression_ A variable that represents a **[TableStyle](Excel.TableStyle.md)** object.
