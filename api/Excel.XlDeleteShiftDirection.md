# XlDeleteShiftDirection Enumeration (Excel)

Specifies how to shift cells to replace deleted cells.



|Name|Value|Description|
|:-----|:-----|:-----|
| **xlShiftToLeft**|-4159|Cells are shifted to the left.|
| **xlShiftUp**|-4162|Cells are shifted up.|
