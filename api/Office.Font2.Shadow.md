# Font2.Shadow property (Office)

Gets the value specifying the type of shadow effect for the selection of text. Read-only.


## Syntax

_expression_.**Shadow**

_expression_ An expression that returns a **[Font2](Office.Font2.md)** object.


## Return value

ShadowFormat
