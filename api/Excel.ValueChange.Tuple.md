# ValueChange.Tuple property (Excel)

Returns the MDX tuple of the value that was changed in the OLAP data source. Read-only.


## Syntax

_expression_.**Tuple**

_expression_ A variable that represents a **[ValueChange](Excel.ValueChange.md)** object.


## Return value

**String**
