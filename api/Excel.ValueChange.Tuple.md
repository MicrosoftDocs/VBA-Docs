# ValueChange.Tuple Property (Excel)

Returns the MDX tuple of the value was changed in the OLAP data source. Read-only


## Syntax

 _expression_. `Tuple`

 _expression_ A variable that represents a '[ValueChange](Excel.ValueChange.md)' object.


### Return value

 **String**
