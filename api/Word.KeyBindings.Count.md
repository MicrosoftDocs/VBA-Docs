# KeyBindings.Count property (Word)

Returns a  **Long** that represents the number of key bindings in the collection. Read-only.


## Syntax

_expression_.**Count**

_expression_ Required. A variable that represents a '[KeyBindings](Word.keybindings.md)' collection.
