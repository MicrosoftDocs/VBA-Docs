# Shape.SoftEdge Property (PowerPoint)

Returns the soft edge format for the specified shape. Read-only.


## Syntax

 _expression_. `SoftEdge`

 _expression_ A variable that represents a [Shape](./PowerPoint.Shape.md) object.


## Return value

MsoSoftEdgeType
