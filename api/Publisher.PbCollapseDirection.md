# PbCollapseDirection Enumeration (Publisher)

The direction in which to collapse the range or selection.



|Name|Value|Description|
|:-----|:-----|:-----|
| **pbCollapseEnd**|2|Collapse at the end.|
| **pbCollapseStart**|1|Collapse at the start.|
