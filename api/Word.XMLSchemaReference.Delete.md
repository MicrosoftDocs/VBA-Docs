# XMLSchemaReference.Delete method (Word)

Deletes the specified XML Schema Reference.


## Syntax

 _expression_. `Delete`

 _expression_ Required. A variable that represents a '[XMLSchemaReference](Word.XMLSchemaReference.md)' object.
