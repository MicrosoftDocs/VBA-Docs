# DropLines.Border property (PowerPoint)

Returns the border of the object. Read-only  **[ChartBorder](PowerPoint.ChartBorder.md)**.


## Syntax

_expression_.**Border**

_expression_ A variable that represents a '[DropLines](PowerPoint.DropLines.md)' object.
