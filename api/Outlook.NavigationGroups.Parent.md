# NavigationGroups.Parent Property (Outlook)

Returns the parent  **Object** of the specified object. Read-only.


## Syntax

_expression_. `Parent`

_expression_ A variable that represents a [NavigationGroups](./Outlook.NavigationGroups.md) object.
