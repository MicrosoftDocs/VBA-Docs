# ChartArea.Shadow property (Excel)

Returns or sets a  **Boolean** value that determines if the object has a shadow.


## Syntax

_expression_. `Shadow`

_expression_ A variable that represents a [ChartArea](Excel.ChartArea-graph-property.md) object.
