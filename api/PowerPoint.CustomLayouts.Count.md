# CustomLayouts.Count property (PowerPoint)

Returns the number of objects in the specified collection. Read-only.


## Syntax

_expression_.**Count**

_expression_ A variable that represents a [CustomLayouts](PowerPoint.CustomLayouts.md) object.


## Return value

Long
