# BulletFormat2.Parent Property (Office)

Gets the parent of the  **BulletFormat2** object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ An expression that returns a [BulletFormat2](./Office.BulletFormat2.md) object.


### Return Value

Object
