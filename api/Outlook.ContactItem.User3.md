# ContactItem.User3 property (Outlook)

Returns or sets a  **String** specifying the third field on the Contacts form intended for miscellaneous use for the contact. Read/write.


## Syntax

_expression_. `User3`

_expression_ A variable that represents a [ContactItem](Outlook.ContactItem.md) object.


## Remarks

The properties  **[ContactItem.User1](Outlook.ContactItem.User1.md)**, **[ContactItem.User2](Outlook.ContactItem.User2.md)**, **ContactItem.User3**, and **[ContactItem.User4](Outlook.ContactItem.User4.md)** map to the fields **User Field 1**,  **User Field 2**,  **User Field 3**, and  **User Field 4** on the Contacts form respectively. These properties are explicit built-in **String** properties; users can use these fields for miscellaneous purposes for the contact.
