# FilterEffect.Type property (PowerPoint)

Represents the type of animation. Read/write.


## Syntax

_expression_.**Type**

_expression_ A variable that represents a [FilterEffect](PowerPoint.FilterEffect.md) object.


## Return value

MsoAnimType


## Remarks

The value of the  **Type** property can be one of these **MsoAnimType** constants.


||
|:-----|
|**MsoAnimTypeColor**|
|**MsoAnimTypeMixed**|
|**MsoAnimTypeMotion**|
|**MsoAnimTypeNone**|
|**MsoAnimTypeProperty**|
|**MsoAnimTypeRotation**|
|**MsoAnimTypeScale**|
|**MsoAnimTypeTransition**|
