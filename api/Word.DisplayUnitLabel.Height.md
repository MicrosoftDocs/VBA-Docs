# DisplayUnitLabel.Height property (Word)

Returns the height, in [points](../language/glossary/vbe-glossary.md#point), of the object. Read-only.


## Syntax

_expression_.**Height**

_expression_ A variable that represents a '[DisplayUnitLabel](Word.DisplayUnitLabel.md)' object.


## Return value

Double
