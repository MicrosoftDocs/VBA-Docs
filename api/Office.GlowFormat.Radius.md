# GlowFormat.Radius Property (Office)

Gets or sets the radius value of the glow effect for the  **GlowFormat** object. Read/write.


## Syntax

 _expression_. `Radius`

 _expression_ An expression that returns a [GlowFormat](./Office.GlowFormat.md) object.


## Return value

Single
