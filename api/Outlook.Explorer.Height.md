# Explorer.Height property (Outlook)

Returns or sets a **Long** specifying the height (in pixels) of the explorer window. Read/write.


## Syntax

_expression_.**Height**

_expression_ A variable that represents an **[Explorer](Outlook.Explorer.md)** object.
