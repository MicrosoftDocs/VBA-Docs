# PickerResult.Fields property (Office)

Represents field definitions of **SubItems** in a **PickerFields** collection. Read-only.


## Syntax

_expression_.**Fields**

_expression_ An expression that returns a **[PickerResult](Office.PickerResult.md)** object.
