# EmailSignatureEntry.Delete method (Word)

Deletes the specified email signature.


## Syntax

 _expression_. `Delete`

 _expression_ Required. A variable that represents an '[EmailSignatureEntry](Word.EmailSignatureEntry.md)' object.
