# BorderArtFormat.Exists Property (Publisher)

 **True** if the specified **BorderArtFormat** object exists. Read-only **Boolean**. 


## Syntax

 _expression_. **Exists**

 _expression_ A variable that represents a  **BorderArtFormat** object.
