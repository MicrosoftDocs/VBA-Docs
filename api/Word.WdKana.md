# WdKana Enumeration (Word)

Specifies the format of the selected Japanese text.



|Name|Value|Description|
|:-----|:-----|:-----|
| **wdKanaHiragana**|9|The text is formatted as Hiragana.|
| **wdKanaKatakana**|8|The text is formatted as Katakana.|
