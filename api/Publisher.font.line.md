# Font.Line Property (Publisher)

Returns a  [LineFormat](Publisher.LineFormat.md) object that specifies the formatting for a line. Read/write.


## Syntax

 _expression_. **Line**

 _expression_ A variable that represents a  **Font** object.


## Return value

 **LINEFORMAT**
