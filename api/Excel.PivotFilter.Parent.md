# PivotFilter.Parent property (Excel)

Returns the parent object for the specified **PivotFilter** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a **[PivotFilter](Excel.PivotFilter.md)** object.
