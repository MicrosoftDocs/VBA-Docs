# CalendarSharing.Parent Property (Outlook)

Returns the parent  **Object** of the specified object. Read-only.


## Syntax

_expression_. `Parent`

_expression_ A variable that represents a [CalendarSharing](./Outlook.CalendarSharing.md) object.
