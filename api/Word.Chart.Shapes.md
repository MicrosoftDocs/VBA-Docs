# Chart.Shapes property (Word)

Returns a collection that represents all the shapes on the chart sheet. Read-only  **[Shapes](Word.shapes.md)**.


## Syntax

 _expression_. `Shapes`

 _expression_ A variable that represents a '[Chart](Word.Chart.md)' object.
