# WorkflowTask.Description Property (Office)

Gets the description of a workflow task. Read-only.


## Syntax

 _expression_. `Description`

 _expression_ An expression that returns a [WorkflowTask](./Office.WorkflowTask.md) object.


### Return Value

String
