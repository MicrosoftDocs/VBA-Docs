# AutoCaptions.Count property (Word)

Returns the number of items in the  **AutoCaptions** collection. Read-only **Long**.


## Syntax

_expression_.**Count**

_expression_ A variable that represents a '[AutoCaptions](Word.autocaptions.md)' object.
