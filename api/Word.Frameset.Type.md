# Frameset.Type property (Word)

Returns the **Frameset** object type. Read-only **[WdFramesetType](Word.WdFramesetType.md)**.


## Syntax

_expression_.**Type**

_expression_ Required. A variable that represents a '[Frameset](Word.Frameset.md)' object.
