# ShapeRange.Rotation Property (Project)
Gets or sets the rotation of the shape range, in degrees. Read/write  **Single**.

## Syntax

 _expression_. `Rotation`

 _expression_ A variable that represents a 'ShapeRange' object.


## Property value

 **FLOAT**
