# CustomLayout.Name Property (PowerPoint)

Represents the display name of the specified custom layout. Read/write.


## Syntax

 _expression_. `Name`

 _expression_ A variable that represents a [CustomLayout](./PowerPoint.CustomLayout.md) object.


### Return value

String
