# Subproject.Index property (Project)

Gets the index of a **Subproject** object in the **Subprojects** containing object. Read-only **Long**.


## Syntax

_expression_.**Index**

_expression_ A variable that represents a [Subproject](./Project.Subproject.md) object.


## Remarks

The **Index** properties of different objects are used in similar ways. For an example, see the **[Index](Project.Project.Index.md)** property of the **Project** object.
