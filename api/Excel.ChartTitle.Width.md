# ChartTitle.Width property (Excel)

Returns the width, in points, of the object. Read-only.


## Syntax

_expression_. `Width`

_expression_ A variable that represents a [ChartTitle](Excel.ChartTitle-graph-property.md) object.


## Return value

Double
