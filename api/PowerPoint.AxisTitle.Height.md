# AxisTitle.Height Property (PowerPoint)

Returns the height, in points, of the object. Read-only


## Syntax

 _expression_. `Height`

_expression_ A variable that represents an [AxisTitle](./PowerPoint.AxisTitle.md) object.


## Return value

Double
