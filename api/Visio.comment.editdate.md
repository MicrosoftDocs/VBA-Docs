# Comment.EditDate Property (Visio)

Returns the most recent date on which the specified comment was edited. Read-only  **Date**.


## Syntax

 _expression_. `EditDate`

 _expression_ A variable that represents a 'Comment' object.


## Property value

 **DATE**
