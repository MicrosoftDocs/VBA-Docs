# Resource.Cost8 property (Project)

Gets or sets the value of the  **Cost8** custom field for the resource. Read/write **Variant**.


## Syntax

_expression_. `Cost8`

_expression_ A variable that represents a [Resource](./Project.Resource.md) object.
