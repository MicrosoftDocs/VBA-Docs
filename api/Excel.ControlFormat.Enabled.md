# ControlFormat.Enabled Property (Excel)

 **True** if the object is enabled. Read/write **Boolean** .


## Syntax

 _expression_. `Enabled`

 _expression_ A variable that represents a [ControlFormat](Excel.ControlFormat.md) object.
