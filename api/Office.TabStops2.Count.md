# TabStops2.Count property (Office)

Gets a  **Long** indicating the number of items in the **TabStops2** collection. Read-only.


## Syntax

_expression_. `Count`

 _expression_ An expression that returns a [TabStops2](Office.TabStops2.md) object.


## Return value

Long
