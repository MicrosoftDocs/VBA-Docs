# Assignment.BCWP Property (Project)

Gets the budgeted cost of work performed for the assignment. Read-only  **Variant**.


## Syntax

 _expression_. `BCWP`

 _expression_ A variable that represents an [Assignment](./Project.Assignment.md) object.
