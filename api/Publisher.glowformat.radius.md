# GlowFormat.Radius Property (Publisher)

Returns or sets a  **Single** that represents the length of the radius for a glow effect. Read/write.


## Syntax

 _expression_. **Radius**

 _expression_ A variable that represents a  **GlowFormat** object.


## Return value

 **Single**
