# PlotArea.ClearFormats method (Excel)

Clears the formatting of the object.


## Syntax

_expression_. `ClearFormats`

_expression_ A variable that represents a [PlotArea](Excel.PlotArea-graph-property.md) object.


## Return value

Variant
