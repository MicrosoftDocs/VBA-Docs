# Application.DDETerminate Method (Project)

Ends a dynamic data exchange (DDE) session.


## Syntax

 _expression_. `DDETerminate`

 _expression_ A variable that represents an [Application](./Project.Application.md) object.


## Return value

 **Boolean**
