# XlColorIndex Enumeration (Excel)

Specifies the color of a selected feature, such as a border, font, or fill.



|Name|Value|Description|
|:-----|:-----|:-----|
| **xlColorIndexAutomatic**|-4105|Automatic color.|
| **xlColorIndexNone**|-4142|No color.|
