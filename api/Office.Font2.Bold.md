# Font2.Bold Property (Office)

Gets or sets a value specifying whether the font should be bold. Read/write.


## Syntax

 _expression_. `Bold`

 _expression_ An expression that returns a [Font2](./Office.Font2.md) object.


### Return Value

MsoTriState
