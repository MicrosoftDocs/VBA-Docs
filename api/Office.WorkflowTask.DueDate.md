# WorkflowTask.DueDate property (Office)

Gets the date that a workflow task is due. Read-only.


## Syntax

_expression_.**DueDate**

_expression_ An expression that returns a **[WorkflowTask](Office.WorkflowTask.md)** object.


## Return value

Date
