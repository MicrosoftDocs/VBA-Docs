# ShapeRange.ThreeD Property (Project)
Gets a  **ThreeDFormat** object that contains 3-D formatting properties for the shape range. Read-only **[ThreeDFormat](https://msdn.microsoft.com/library/office/ff836783%28v=office.15%29)**.

## Syntax

 _expression_. `ThreeD`

 _expression_ A variable that represents a 'ShapeRange' object.


## Remarks


 **Note**  You cannot apply 3-D formatting to some kinds of shapes, such as beveled shapes or multiple disjointed paths. Most of the properties and methods of the  **ThreeDFormat** object fail for such shapes.


## Property value

 **THREEDFORMAT**
