# OlkDateControl.Change Event (Outlook)

Occurs when the  **[Value](Outlook.OlkDateControl.Value.md)** property changes.


## Syntax

_expression_. `Change`

_expression_ A variable that represents an [OlkDateControl](./Outlook.OlkDateControl.md) object.
