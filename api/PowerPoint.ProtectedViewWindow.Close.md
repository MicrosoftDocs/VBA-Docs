# ProtectedViewWindow.Close Method (PowerPoint)

Closes the specified window.


## Syntax

 _expression_. `Close`

 _expression_ A variable that represents a [ProtectedViewWindow](./PowerPoint.ProtectedViewWindow.md) object.
