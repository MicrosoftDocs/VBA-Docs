# SparklineGroups.Clear Method (Excel)

Clears the selected sparklines.


## Syntax

 _expression_. `Clear`

 _expression_ A variable that represents a '[SparklineGroups](Excel.SparklineGroups.md)' object.


### Return value

Nothing
