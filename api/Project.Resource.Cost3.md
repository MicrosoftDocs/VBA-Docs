# Resource.Cost3 property (Project)

Gets or sets the value of the  **Cost3** custom field for the resource. Read/write **Variant**.


## Syntax

_expression_. `Cost3`

_expression_ A variable that represents a [Resource](./Project.Resource.md) object.
