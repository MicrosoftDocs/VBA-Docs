# Axis.Width property (Word)

Returns the width, in [points](../language/glossary/vbe-glossary.md#point), of the object. Read-only  **Double**.


## Syntax

_expression_.**Width**

_expression_ A variable that represents an **[Axis](Word.Axis.md)** object.
