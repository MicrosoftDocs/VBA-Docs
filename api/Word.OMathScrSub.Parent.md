# OMathScrSub.Parent property (Word)

Returns an **Object** that represents the parent object of the specified **OMathScrSub** object.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a '[OMathScrSub](Word.OMathScrSub.md)' object.
