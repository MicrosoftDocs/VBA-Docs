# TextFrame2.Parent property (PowerPoint)

Returns the parent object of the specified test frame. Read-only.


## Syntax

_expression_.**Parent**

 _expression_ An expression that returns a **[TextFrame2](PowerPoint.TextFrame2.md)** object.


## Return value

Object
