# ShapeRange.ApplyAnimation method (PowerPoint)

Applies animation to a  **ShapeRange** object.


## Syntax

_expression_. `ApplyAnimation`

_expression_ A variable that represents a **[ShapeRange](PowerPoint.ShapeRange.md)** object.
