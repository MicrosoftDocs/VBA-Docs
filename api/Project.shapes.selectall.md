# Shapes.SelectAll method (Project)
Selects all shapes in the **Shapes** collection.

## Syntax

_expression_.**SelectAll**

_expression_ A variable that represents a **[Shapes](Project.Shapes.md)** object.


## Return value

 **Nothing**
