# AxisTitle.Shadow property (Word)

Returns or sets a value that determines whether the object has a shadow. Read/write  **Boolean**.


## Syntax

_expression_.**Shadow**

_expression_ A variable that represents an '[AxisTitle](Word.AxisTitle.md)' object.
