# XmlMap object (Excel)

Represents an XML map that has been added to a workbook.


## Remarks

Use the  **[Add](Excel.XmlMaps.Add.md)** method of the **[XmlMaps](Excel.XmlMaps.md)** collection to add an XML map to a workbook.


### Importing and exporting XML data

Use the  **[Import](Excel.XmlMap.Import.md)** method to import XML data from an XML data file into cells mapped to the specified **XmlMap**. The **[ImportXml](Excel.XmlMap.ImportXml.md)** method imports XML data for a **String** variable.

Use the  **[Export](Excel.XmlMap.Export.md)** method to export data from cells mapped to the specified **XmlMap**. The **[ExportXml](Excel.XmlMap.ExportXml.md)** method exports XML data to a **String** variable.
