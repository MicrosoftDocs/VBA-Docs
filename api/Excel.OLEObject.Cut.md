# OLEObject.Cut Method (Excel)

Cuts the object to the Clipboard or pastes it into a specified destination.


## Syntax

 _expression_. `Cut`

 _expression_ A variable that represents an [OLEObject](Excel.OLEObject.md) object.


### Return Value

Variant
