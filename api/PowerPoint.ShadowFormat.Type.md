# ShadowFormat.Type Property (PowerPoint)

Represents the type of shadow. Read/write.


## Syntax

 _expression_. `Type`

 _expression_ A variable that represents a [ShadowFormat](./PowerPoint.ShadowFormat.md) object.


## Return value

MsoShadowType
