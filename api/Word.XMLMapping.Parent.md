# XMLMapping.Parent property (Word)

Returns an **Object** that represents the parent object of the specified **XMLMapping** object.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a '[XMLMapping](Word.XMLMapping.md)' object.
