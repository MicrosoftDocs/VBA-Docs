# FullSeriesCollection.Count Property (PowerPoint)

Returns a  **Long** that represents the number of full series in the collection. Read-only.


## Syntax

 _expression_. `Count`

 _expression_ A variable that represents a 'FullSeriesCollection' object.


## Property value

 **INT32**
