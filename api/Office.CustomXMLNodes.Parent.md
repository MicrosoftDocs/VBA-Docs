# CustomXMLNodes.Parent property (Office)

Gets the  **Parent** object for the **CustomXMLNodes** object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ An expression that returns a [CustomXMLNodes](Office.CustomXMLNodes.md) object.


## Return value

Object
