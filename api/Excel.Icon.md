# Icon Object (Excel)

Represents a single icon in an icon set used in a conditional formatting rule.


## Remarks

The  **Icon** object is returned from the **[Item](Excel.IconSet.Item.md)** property of the **[IconSet](Excel.IconSet.md)** object.
