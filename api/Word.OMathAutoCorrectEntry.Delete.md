# OMathAutoCorrectEntry.Delete method (Word)

Deletes the specified equation auto correct entry.


## Syntax

 _expression_. `Delete`

 _expression_ An expression that returns an [OMathAutoCorrectEntry](./Word.OMathAutoCorrectEntry.md) object.
