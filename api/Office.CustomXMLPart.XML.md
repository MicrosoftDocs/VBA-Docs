# CustomXMLPart.XML Property (Office)

Gets the XML representation of the current  **CustomXMLPart** object. Read-only.


## Syntax

 _expression_. `XML`

 _expression_ An expression that returns a [CustomXMLPart](./Office.CustomXMLPart.md) object.


### Return Value

String
