# Project.GetServerProjectGuid Method (Project)

Returns the GUID for the enterprise project.


## Syntax

 _expression_. `GetServerProjectGuid`

 _expression_ A variable that represents a [Project](./Project(enumerations).md) object.


## Return value

 **String**
