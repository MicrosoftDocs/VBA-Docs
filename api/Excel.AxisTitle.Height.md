# AxisTitle.Height property (Excel)

Returns the height, in points, of the object. Read-only.


## Syntax

_expression_. `Height`

_expression_ A variable that represents an [AxisTitle](Excel.AxisTitle-graph-property.md) object.


## Return value

Double
