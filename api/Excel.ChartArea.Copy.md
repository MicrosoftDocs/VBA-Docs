# ChartArea.Copy Method (Excel)

Copies the object to the Clipboard.


## Syntax

 _expression_. `Copy`

 _expression_ A variable that represents a [ChartArea](Excel.ChartArea(Graph property).md) object.


### Return Value

Variant
