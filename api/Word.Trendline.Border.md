# Trendline.Border property (Word)

Returns the border of the object. Read-only  **[ChartBorder](Word.ChartBorder.md)**.


## Syntax

_expression_.**Border**

_expression_ A variable that represents a '[Trendline](Word.Trendline.md)' object.
