# AxisTitle.Select method (Excel)

Selects the object.


## Syntax

_expression_.**Select**

_expression_ A variable that represents an **[AxisTitle](Excel.AxisTitle(object).md)** object.


## Return value

A **Variant** value representing the selected object.
