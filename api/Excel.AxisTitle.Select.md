# AxisTitle.Select Method (Excel)

Selects the object.


## Syntax

 _expression_. `Select`

 _expression_ A variable that represents an [AxisTitle](Excel.AxisTitle(Graph property).md) object.


### Return value

A Variant value representing the selected object.
