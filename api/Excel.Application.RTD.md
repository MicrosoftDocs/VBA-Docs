# Application.RTD Property (Excel)

Returns an  **[RTD](Excel.RTD.md)** object.


## Syntax

 _expression_. `RTD`

 _expression_ An expression that returns a [Application](Excel.Application(Graph property).md) object.


### Return Value

RTD
