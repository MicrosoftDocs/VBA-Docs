# PjCalendarBarType Enumeration (Project)

Contains constants that specify a calendar bar type.



|Name|Value|Description|
|:-----|:-----|:-----|
|**pjLineBar**|1|Line bar.|
|**pjNoBar**|2|No bar.|
|**pjNormalBar**|0|Normal bar.|
