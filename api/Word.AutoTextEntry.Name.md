# AutoTextEntry.Name property (Word)

Returns or sets the name of the specified object.Read/write  **String**.


## Syntax

 _expression_. `Name`

 _expression_ A variable that represents a '[AutoTextEntry](Word.AutoTextEntry.md)' object.
