# OfficeTheme.Parent property (Office)

Gets the  **Parent** object for the **OfficeTheme** object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ An expression that returns a [OfficeTheme](Office.OfficeTheme.md) object.


## Return value

Object
