# PivotLineCells.Count Property (Excel)

Returns the number of items in the  **PivotLineCells** collection. Read-only.


## Syntax

 _expression_. `Count`

 _expression_ A variable that represents a [PivotLineCells](Excel.PivotLineCells.md) object.
