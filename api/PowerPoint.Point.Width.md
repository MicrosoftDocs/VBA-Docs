# Point.Width property (PowerPoint)

Returns the width, in [points](../language/glossary/vbe-glossary.md#point), of the object. Read-only.


## Syntax

_expression_.**Width**

_expression_ A variable that represents a [Point](PowerPoint.Point.md) object.


## Return value

Double
