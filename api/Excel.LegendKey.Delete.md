# LegendKey.Delete method (Excel)

Deletes the object.


## Syntax

_expression_.**Delete**

_expression_ A variable that represents a **[LegendKey](excel.legendkey(object).md)** object.


## Return value

Variant


## Remarks

Deleting a **LegendKey** object deletes the entire series.
