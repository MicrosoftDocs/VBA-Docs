# KeysBoundTo.Count property (Word)

Returns a  **Long** that represents the number of keys in the collection. Read-only.


## Syntax

 _expression_. `Count`

 _expression_ Required. A variable that represents a '[KeysBoundTo](Word.keysboundto.md)' object.
