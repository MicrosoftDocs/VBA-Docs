# HasLegend Property

 **True** if the chart has a legend. Read/write **Boolean**.


## Example

This example turns on the legend for the chart and then sets the legend font color to blue.


```vb
With myChart 
 .HasLegend = True 
 .Legend.Font.ColorIndex = 5 
End With
```
