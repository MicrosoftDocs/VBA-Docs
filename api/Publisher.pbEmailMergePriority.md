# pbEmailMergePriority Enumeration (Publisher)

Constants for merged e-mail message priority passed to and returned by the  ** [EmailMergeEnvelope.Priority](Publisher.EmailMergeEnvelope.Priority.md)** property.



|Name|Value|Description|
|:-----|:-----|:-----|
| **pbPriorityHigh**|1|High priority|
| **pbPriorityLow**|2|Low priority|
| **pbPriorityNone**|0|No priority set|
