# Shape.Glow property (Word)

Returns a  **[GlowFormat](Word.GlowFormat.md)** object that represents the glow formatting for a shape. Read-only.


## Syntax

 _expression_. `Glow`

 _expression_ An expression that returns a '[Shape](Word.Shape.md)' object.
