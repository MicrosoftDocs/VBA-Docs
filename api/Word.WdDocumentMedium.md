# WdDocumentMedium Enumeration (Word)

Specifies the type of document to which you are applying a theme.



|Name|Value|Description|
|:-----|:-----|:-----|
| **wdDocument**|1|Document.|
| **wdEmailMessage**|0|E-mail message.|
| **wdWebPage**|2|Web page.|
