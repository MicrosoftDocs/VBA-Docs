# XlCutCopyMode Enumeration (Excel)

Specifies whether status is Copy mode or Cut mode.



|Name|Value|Description|
|:-----|:-----|:-----|
| **xlCopy**|1|In Copy mode|
| **xlCut**|2|In Cut mode|
