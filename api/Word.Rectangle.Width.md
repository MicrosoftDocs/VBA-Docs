# Rectangle.Width property (Word)

Returns or sets a  **Long** that represents the width, in points, of a rectangle. Read/write **Long**.


## Syntax

 _expression_. `Width`

 _expression_ A variable that represents a '[Rectangle](Word.Rectangle.md)' object.
