# ThemeFonts.Count property (Office)

Gets a **Long** indicating the number of items in the **ThemeFonts** collection. Read-only.


## Syntax

_expression_.**Count**

_expression_ An expression that returns a **[ThemeFonts](Office.ThemeFonts.md)** object.


## Return value

Integer
