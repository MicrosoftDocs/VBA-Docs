# MetaProperties.Parent Property (Office)

Gets the  **Parent** object for the **MetaProperties** object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ An expression that returns a [MetaProperties](./Office.MetaProperties.md) object.


### Return Value

Object
