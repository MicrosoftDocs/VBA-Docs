# Parameter.Type property (Excel)

Returns an **[XlParameterType](Excel.XlParameterType.md)** value that represents the parameter type.


## Syntax

_expression_.**Type**

_expression_ A variable that represents a **[Parameter](Excel.Parameter.md)** object.
