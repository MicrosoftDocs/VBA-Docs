# TaskItem.Body property (Outlook)

Returns or sets a **String** representing the clear-text body of the Outlook item. Read/write.


## Syntax

_expression_. `Body`

_expression_ A variable that represents a [TaskItem](Outlook.TaskItem.md) object.
