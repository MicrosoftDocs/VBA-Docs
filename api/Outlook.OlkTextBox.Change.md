# OlkTextBox.Change event (Outlook)

Occurs when the  **[Value](Outlook.OlkTextBox.Value.md)** property changes.


## Syntax

_expression_. `Change`

_expression_ A variable that represents an [OlkTextBox](Outlook.OlkTextBox.md) object.
