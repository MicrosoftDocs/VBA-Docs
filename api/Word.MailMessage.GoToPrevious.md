# MailMessage.GoToPrevious method (Word)

Displays the previous mail message if you are using Word as your email editor.


## Syntax

 _expression_. `GoToPrevious`

 _expression_ Required. A variable that represents a '[MailMessage](Word.MailMessage.md)' object.
