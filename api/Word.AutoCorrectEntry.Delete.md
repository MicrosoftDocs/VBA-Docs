# AutoCorrectEntry.Delete method (Word)

Deletes the specified AutoCorrect entry.


## Syntax

 _expression_. `Delete`

 _expression_ Required. A variable that represents an '[AutoCorrectEntry](Word.AutoCorrectEntry.md)' object.
