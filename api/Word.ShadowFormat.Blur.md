# ShadowFormat.Blur property (Word)

Returns or sets a  **Single** that represents the blur level for a shadow format. Read/write.


## Syntax

 _expression_. `Blur`

 _expression_ An expression that returns a '[ShadowFormat](Word.ShadowFormat.md)' object.
