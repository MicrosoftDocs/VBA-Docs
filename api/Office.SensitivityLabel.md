# SensitivityLabel object (Office)

Represents a wrapper object for accessing sensitivity label on the active document.

## Remarks

**SensitivityLabel** applied on a document requires the use of policy defined by organization administrator. The organization is identified by using an identity of an Office Account signed into Office.

> [!NOTE]
> Your organization may need [specific license](/office365/servicedescriptions/microsoft-365-service-descriptions/microsoft-365-tenantlevel-services-licensing-guidance/microsoft-365-security-compliance-licensing-guidance) to get access to the sensitivity label related functionality.
