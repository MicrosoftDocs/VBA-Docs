# Shape.ReverseEnds Method (Visio)

Reverses an object by flipping it both horizontally and vertically.


## Syntax

 _expression_. `ReverseEnds`

 _expression_ A variable that represents a [Shape](./Visio.Shape.md) object.


### Return value

Nothing
