# SharingItem.Copy method (Outlook)

Creates another instance of a  **[SharingItem](Outlook.SharingItem.md)**.


## Syntax

_expression_.**Copy**

_expression_ A variable that represents a [SharingItem](Outlook.SharingItem.md) object.
