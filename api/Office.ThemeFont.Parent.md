# ThemeFont.Parent Property (Office)

Gets the  **Parent** object for the **ThemeFont** object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ An expression that returns a [ThemeFont](./Office.ThemeFont.md) object.


### Return value

Object
