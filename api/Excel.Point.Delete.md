# Point.Delete method (Excel)

Deletes the series the point belongs to.


## Syntax

_expression_.**Delete**

_expression_ A variable that represents a **[Point](Excel.Point(object).md)** object.


## Return value

Variant


## Remarks

Deleting a **Point** object deletes the entire series.
