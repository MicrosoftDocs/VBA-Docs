# CustomTaskPane.Height Property (Office)

Gets or sets the height of the  **CustomTaskPane** object (in points). Read/write.


## Syntax

 _expression_. `Height`

 _expression_ An expression that returns a [CustomTaskPane](./Office.CustomTaskPane.md) object.


### Return value

Integer
