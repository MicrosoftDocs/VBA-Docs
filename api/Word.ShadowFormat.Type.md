# ShadowFormat.Type property (Word)

Returns or sets the shape shadow type. Read/write  **MsoShadowType**.


## Syntax

_expression_.**Type**

_expression_ Required. A variable that represents a **[ShadowFormat](Word.ShadowFormat.md)** object.
