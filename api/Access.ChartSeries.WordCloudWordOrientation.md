# ChartSeries.WordCloudWordOrientation property (Access)

Returns or sets the word orientation for a Word Cloud Modern Chart (only available in Current Channel). Read/write **[AcWordCloudWordOrientation](Access.AcWordCloudWordOrientation.md)**.


## Syntax

_expression_.**WordCloudWordOrientation**

_expression_ A variable that represents a **[ChartSeries](Access.ChartSeries.md)** object.
