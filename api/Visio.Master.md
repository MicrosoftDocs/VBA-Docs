# Master object (Visio)

Represents a master in a stencil.


## Remarks

You retrieve a particular **Master** object from the **[Masters](visio.masters.md)** collection of a **Document** object whose stencil contains that master.

The default property of a **Master** object is **Name**.

To create an instance of a master in a drawing, use the **[Drop](visio.page.drop.md)** method of a **Page** object that represents a drawing page.
