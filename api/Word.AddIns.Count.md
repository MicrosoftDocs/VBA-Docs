# AddIns.Count property (Word)

Returns the number of  **[AddIn](Word.AddIn.md)** objects in the **AddIns** collection. Read-only **Long**.


## Syntax

_expression_.**Count**

 _expression_ An expression that returns an '[AddIns](Word.addins.md)' collection.
