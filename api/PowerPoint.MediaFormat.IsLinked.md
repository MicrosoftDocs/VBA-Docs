# MediaFormat.IsLinked Property (PowerPoint)

Returns whether the media file is linked. Read-only.


## Syntax

 _expression_. `IsLinked`

 _expression_ An expression that returns a [MediaFormat](./PowerPoint.MediaFormat.md) object.


## Return value

Boolean
