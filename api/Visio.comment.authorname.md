# Comment.AuthorName property (Visio)

Returns the name of the author of the comment. Read-only  **String**.


## Syntax

_expression_.**AuthorName**

_expression_ A variable that represents a **[Comment](Visio.Comment.md)** object.


## Property value

 **STRING**
