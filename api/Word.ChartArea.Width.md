# ChartArea.Width property (Word)

Returns or sets the width, in [points](../language/glossary/vbe-glossary.md#point), of the object. Read/write  **Double**.


## Syntax

_expression_.**Width**

_expression_ A variable that represents a '[ChartArea](Word.ChartArea.md)' object.
