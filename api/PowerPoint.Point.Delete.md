# Point.Delete method (PowerPoint)

Deletes the object.


## Syntax

_expression_.**Delete**

_expression_ A variable that represents a '[Point](PowerPoint.Point.md)' object.


## Remarks

Deleting a  **Point** object deletes the entire series.
