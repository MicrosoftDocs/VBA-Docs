# EmptyCell.GridlineWidthTop property (Access)

Gets or sets the width of the top gridline for the specified cell. Read/write **Byte**.


## Syntax

_expression_.**GridlineWidthTop**

_expression_ A variable that represents an **[EmptyCell](Access.EmptyCell.md)** object.


## Remarks

The **GridlineWidthTop** property uses the following settings.

|Setting|Description|
|:-----|:-----|
|0| Hairline. This is the narrowest border possible on your system.|
|1 to 6|The width as indicated in points. The default setting is 1 point.|
