# Exceptions.Count Property (Outlook)

Returns a  **Long** indicating the count of objects in the specified collection. Read-only.


## Syntax

_expression_. `Count`

_expression_ A variable that represents an [Exceptions](./Outlook.Exceptions.md) object.
