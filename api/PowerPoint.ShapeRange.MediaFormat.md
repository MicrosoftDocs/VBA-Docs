# ShapeRange.MediaFormat property (PowerPoint)

Returns the current  **MediaFormat** object. Read-only.


## Syntax

_expression_. `MediaFormat`

_expression_ A variable that represents a **[ShapeRange](PowerPoint.ShapeRange.md)** object.
