# ServerPolicy.Name Property (Office)

Gets the name of the  **ServerPolicy** object. Read-only.


## Syntax

 _expression_. `Name`

 _expression_ An expression that returns a [ServerPolicy](./Office.ServerPolicy.md) object.


### Return Value

String
