# WdFootnoteLocation enumeration (Word)

Specifies the position of footnotes.



|Name|Value|Description|
|:-----|:-----|:-----|
| **wdBeneathText**|1|Beneath current text.|
| **wdBottomOfPage**|0|At bottom of current page.|
