# Application.GanttBarTextDateFormat method (Project)

Sets the date format for text around bars on the Gantt Chart.


## Syntax

_expression_. `GanttBarTextDateFormat`( `_DateFormat_` )

_expression_ A variable that represents an **[Application](Project.Application.md)** object.


## Parameters



|Name|Required/Optional|Data type|Description|
|:-----|:-----|:-----|:-----|
| _DateFormat_|Required|**Long**|A constant specifying the format of dates in the Gantt bar text. Can be one of the following  **[PjDateFormat](Project.PjDateFormat.md)** constants.|

## Return value

 **Boolean**
