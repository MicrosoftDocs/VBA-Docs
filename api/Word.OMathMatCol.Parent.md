# OMathMatCol.Parent property (Word)

Returns an **Object** that represents the parent object of the specified **OMathMatCol** object.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a '[OMathMatCol](Word.OMathMatCol.md)' object.
