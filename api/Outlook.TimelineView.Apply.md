# TimelineView.Apply Method (Outlook)

Applies the  **[TimelineView](Outlook.TimelineView.md)** object to the current view.


## Syntax

 _expression_. `Apply`

_expression_ A variable that represents a [TimelineView](./Outlook.TimelineView.md) object.
