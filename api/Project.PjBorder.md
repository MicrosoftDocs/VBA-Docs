# PjBorder enumeration (Project)

Contains constants that specify the type borders.



|Name|Value|Description|
|:-----|:-----|:-----|
|**pjAroundEveryPage**|1|Around every page.|
|**pjNoBorder**|0|No border.|
|**pjOutsidePages**|2|On the outside pages.|
