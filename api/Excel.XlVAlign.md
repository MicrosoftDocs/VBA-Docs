Specifies the vertical alignment for the object.

|:-----|:-----|:-----|
| **xlVAlignBottom**|-4107|Bottom|
| **xlVAlignCenter**|-4108|Center|
| **xlVAlignDistributed**|-4117|Distributed|
| **xlVAlignJustify**|-4130|Justify|
| **xlVAlignTop**|-4160|Top|
