# SlicerPivotTables.Count property (Excel)

Returns the number of objects in the collection.


## Syntax

_expression_.**Count**

_expression_ A variable that returns a **[SlicerPivotTables](Excel.SlicerPivotTables.md)** object.


## Return value

**Long**
