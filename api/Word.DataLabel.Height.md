# DataLabel.Height Property (Word)

Gets or sets the height, in points, of the object. Read/write  **Double**.


## Syntax

 _expression_. `Height`

 _expression_ A variable that represents a '[DataLabel](Word.DataLabel.md)' object.


### Return value

Double
