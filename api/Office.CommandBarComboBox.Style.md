# CommandBarComboBox.Style property (Office)

Gets or sets the way a **CommandBarComboBox** control is displayed. Can be either of the following **[msoComboStyle](office.msocombostyle.md)** constants: **msoComboLabel** or **msoComboNormal**. Read/write.

> [!NOTE] 
> The use of CommandBars in some Microsoft Office applications has been superseded by the new ribbon component of the Microsoft Office Fluent user interface. For more information, see [Overview of the Office Fluent ribbon](../library-reference/concepts/overview-of-the-office-fluent-ribbon.md).


## Syntax

_expression_.**Style**

_expression_ A variable that represents a **[CommandBarComboBox](Office.CommandBarComboBox.md)** object.
