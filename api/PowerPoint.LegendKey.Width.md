# LegendKey.Width Property (PowerPoint)

Returns the width, in points, of the object. Read-only  **Double**.


## Syntax

 _expression_. `Width`

 _expression_ A variable that represents a '[LegendKey](PowerPoint.LegendKey.md)' object.
