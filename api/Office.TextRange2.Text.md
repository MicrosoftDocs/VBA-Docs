# TextRange2.Text property (Office)

Gets or sets a **String** value that represents the text in a text range. Read/write.


## Syntax

_expression_.**Text**

_expression_ An expression that returns a **[TextRange2](Office.TextRange2.md)** object.


## Return value

String
