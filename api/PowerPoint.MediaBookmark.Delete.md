# MediaBookmark.Delete Method (PowerPoint)

Deletes the bookmark from the collection. 


## Syntax

 _expression_. `Delete`

 _expression_ A variable that represents a [MediaBookmark](./PowerPoint.MediaBookmark.md) object.


## Return value

None
