# Comments.Document Property (Visio)

Returns the [Document](Visio.Document.md) object that is associated with an object. Read-only.


## Syntax

 _expression_. `Document`

 _expression_ A variable that represents a 'Comments' object.


## Property value

 **IVDOCUMENT**
