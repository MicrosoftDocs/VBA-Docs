# Font2.Italic Property (Office)

Gets or sets a value specifying whether the text for a selection is italic. Read/write.


## Syntax

 _expression_. `Italic`

 _expression_ An expression that returns a [Font2](./Office.Font2.md) object.


## Return value

MsoTriState
