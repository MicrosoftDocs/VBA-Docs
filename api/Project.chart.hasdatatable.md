# Chart.HasDataTable property (Project)
 **True** if the chart has a data table. Read/write **Boolean**.

## Syntax

_expression_.**HasDataTable**

_expression_ A variable that represents a **[Chart](Project.Chart.md)** object.


## Property value

 **BOOL**
