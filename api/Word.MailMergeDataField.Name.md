# MailMergeDataField.Name property (Word)

Returns name of the specified object. Read-only  **String**.


## Syntax

 _expression_. `Name`

 _expression_ Required. A variable that represents a '[MailMergeDataField](Word.MailMergeDataField.md)' object.
