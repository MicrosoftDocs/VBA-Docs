# BuildingBlock.Index property (Word)

Returns a  **Long** that represents the position of an item in a collection. Read-only.


## Syntax

_expression_.**Index**

 _expression_ An expression that returns a [BuildingBlock](./Word.BuildingBlock.md) object.
