# SlicerItem.SourceName property (Excel)

Returns the source name of the slicer item. Read-only.


## Syntax

_expression_. `SourceName`

_expression_ A variable that represents a '[SlicerItem](Excel.SlicerItem.md)' object.


## Return value

Variant
