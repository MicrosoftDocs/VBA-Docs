# ChartColorFormat.Parent property (Word)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a '[ChartColorFormat](Word.ChartColorFormat.md)' object.
