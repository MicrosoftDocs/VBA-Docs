# Series.Copy method (Excel)

If the series has a picture fill, then this method copies the picture to the Clipboard.


## Syntax

_expression_.**Copy**

_expression_ A variable that represents a [Series](Excel.Series-graph-object.md) object.


## Return value

Variant
