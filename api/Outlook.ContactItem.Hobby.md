# ContactItem.Hobby property (Outlook)

Returns or sets a  **String** representing the hobby for the contact. Read/write.


## Syntax

_expression_. `Hobby`

_expression_ A variable that represents a [ContactItem](Outlook.ContactItem.md) object.
