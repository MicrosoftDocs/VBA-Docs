# Comment.Collapsed Property (Visio)

Returns or sets whether the specified comment is collapsed. Read-write  **Boolean**.


## Syntax

 _expression_. `Collapsed`

 _expression_ A variable that represents a 'Comment' object.


## Property value

 **BOOL**
