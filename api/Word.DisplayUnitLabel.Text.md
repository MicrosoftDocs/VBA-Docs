# DisplayUnitLabel.Text property (Word)

Returns or sets the text for the specified object. Read/write  **String**.


## Syntax

_expression_.**Text**

_expression_ A variable that represents a '[DisplayUnitLabel](Word.DisplayUnitLabel.md)' object.
