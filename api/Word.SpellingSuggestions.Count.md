# SpellingSuggestions.Count property (Word)

Returns a  **Long** that represents the number of spelling suggestions in the collection. Read-only.


## Syntax

_expression_.**Count**

_expression_ Required. A variable that represents a '[SpellingSuggestions](Word.spellingsuggestions.md)' collection.
