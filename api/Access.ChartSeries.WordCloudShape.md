# ChartSeries.WordCloudShape property (Access)

Returns or sets the shape for a Word Cloud Modern Chart (only available in Current Channel). Read/write **[AcWordCloudShape](Access.AcWordCloudShape.md)**.


## Syntax

_expression_.**WordCloudShape**

_expression_ A variable that represents a **[ChartSeries](Access.ChartSeries.md)** object.
