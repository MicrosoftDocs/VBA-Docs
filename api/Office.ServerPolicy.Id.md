# ServerPolicy.Id Property (Office)

Gets the ID of a server policy. Read-only.


## Syntax

 _expression_. `Id`

 _expression_ An expression that returns a [ServerPolicy](./Office.ServerPolicy.md) object.


### Return Value

String
