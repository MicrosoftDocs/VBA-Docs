# Shape.Chart property (Excel)

Returns a  **[Chart](Excel.Chart(object).md)** object that represents the chart contained in the shape. Read-only.


## Syntax

_expression_. `Chart`

_expression_ A variable that represents a [Shape](./Excel.Shape.md) object.
