# Shape.Table property (Project)
Gets the **ReportTable** object in the shape. Read-only[ReportTable](Project.reporttable.md).

## Syntax

_expression_. `Table`

_expression_ A variable that represents a **[Shape](Project.Shape.md)** object.


## Property value

 **REPORTTABLE**
