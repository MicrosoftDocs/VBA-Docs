# Axis.Format property (Word)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](Word.ChartFormat.md)**.


## Syntax

_expression_.**Format**

_expression_ A variable that represents an **[Axis](Word.Axis.md)** object.
