# AxisTitle.Width property (Word)

Returns the width, in [points](../language/glossary/vbe-glossary.md#point), of the object. Read-only.


## Syntax

_expression_.**Width**

_expression_ A variable that represents an '[AxisTitle](Word.AxisTitle.md)' object.
