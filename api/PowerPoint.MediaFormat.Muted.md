# MediaFormat.Muted property (PowerPoint)

Returns whether audio playback of the media is muted. Read/write


## Syntax

_expression_. `Muted`

 _expression_ An expression that returns a [MediaFormat](PowerPoint.MediaFormat.md) object.


## Return value

Boolean
