# AxisTitle.Width property (Excel)

Returns the width, in points, of the object. Read-only.


## Syntax

_expression_. `Width`

_expression_ A variable that represents an [AxisTitle](Excel.AxisTitle-graph-property.md) object.


## Return value

Double
