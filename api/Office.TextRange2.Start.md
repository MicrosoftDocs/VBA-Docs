# TextRange2.Start property (Office)

Gets a **Long** value indicating the starting point of the specified text range. Read-only.


## Syntax

_expression_.**Start**

_expression_ An expression that returns a **[TextRange2](Office.TextRange2.md)** object.


## Return value

Long
