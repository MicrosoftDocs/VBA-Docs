# PjLayoutMode Enumeration (Project)

Contains constants that specify the layout mode.



|Name|Value|Description|
|:-----|:-----|:-----|
|**pjLayoutAutomatic**|1|Layout is done automatically.|
|**pjLayoutManual**|2|Layout is done manually.|
