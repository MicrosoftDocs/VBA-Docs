# ModelTable.Name property (Excel)

Returns a  **String** value that represents the name of the object.


## Syntax

_expression_. `Name`

_expression_ A variable that represents a [ModelTable object (Excel)](Excel.modeltable.md) object.


## Property value

 **STRING**
