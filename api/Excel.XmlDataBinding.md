# XmlDataBinding object (Excel)

Represents the connection to the source data for an  **[XmlMap](Excel.XmlMap.md)** object.


## Remarks

Use the  **[LoadSettings](Excel.XmlDataBinding.LoadSettings.md)** method initialize the settings for an **XmlDataBinding** object

Use the  **[Refresh](Excel.XmlDataBinding.Refresh.md)** method to refresh a data binding.

Use the  **[ClearSettings](Excel.XmlDataBinding.ClearSettings.md)** method to remove a data binding.
