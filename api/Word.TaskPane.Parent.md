# TaskPane.Parent property (Word)

Returns an **Object** that represents the parent object of the specified **TaskPane** object.


## Syntax

_expression_.**Parent**

_expression_ Required. A variable that represents a '[TaskPane](Word.TaskPane.md)' object.
