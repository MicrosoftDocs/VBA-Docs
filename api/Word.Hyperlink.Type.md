# Hyperlink.Type property (Word)

Returns the hyperlink type. Read-only  **MsoHyperlinkType**.


## Syntax

_expression_.**Type**

_expression_ Required. A variable that represents a '[Hyperlink](Word.Hyperlink.md)' object.
