# COMAddIn.Parent property (Office)

Gets the **Parent** object for the **COMAddIn** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a **[COMAddIn](Office.COMAddIn.md)** object.


## Return value

Object
