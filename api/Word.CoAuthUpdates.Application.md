# CoAuthUpdates.Application property (Word)

Returns an [Application](Word.Application.md) object that represents the Microsoft Word application. Read-only.


## Syntax

 _expression_. `Application`

 _expression_ An expression that returns a 'CoAuthUpdates' object.
