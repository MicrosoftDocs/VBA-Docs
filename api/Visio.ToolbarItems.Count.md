# ToolbarItems.Count Property (Visio)

Returns the number of objects in a collection. Read-only.


## Syntax

 _expression_. `Count`

 _expression_ A variable that represents a [ToolbarItems](./Visio.ToolbarItems.md) object.


## Return value

Long
