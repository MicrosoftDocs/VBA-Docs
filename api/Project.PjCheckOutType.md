# PjCheckOutType Enumeration (Project)

Contains constants that specify the type of checkout.



|Name|Value|Description|
|:-----|:-----|:-----|
|**pjReadOnly**|0|Check out as read-only.|
|**pjReadWrite**|1|Check out as read/write.|
