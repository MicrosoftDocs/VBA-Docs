# DataLabel.Width Property (Excel)

Returns the width, in points, of the object. Read-only.


## Syntax

 _expression_. `Width`

 _expression_ A variable that represents a [DataLabel](Excel.DataLabel(Graph property).md) object.


### Return Value

Double
