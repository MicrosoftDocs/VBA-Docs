# WorkflowTask.Name property (Office)

Gets the name of the  **WorkflowTask** object. Read-only.


## Syntax

_expression_. `Name`

 _expression_ An expression that returns a [WorkflowTask](Office.WorkflowTask.md) object.


## Return value

String
