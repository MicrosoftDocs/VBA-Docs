# Cell.IsConstant Property (Visio)

Determines whether a formula of the cell is a constant expression. Read-only.


## Syntax

 _expression_. `IsConstant`

 _expression_ A variable that represents a [Cell](./Visio.Cell.md) object.


## Return value

Integer
