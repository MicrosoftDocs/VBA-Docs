# MediaFormat.Length property (PowerPoint)

Returns the total length of the media in time units. Read-only.


## Syntax

_expression_.**Length**

 _expression_ An expression that returns a [MediaFormat](PowerPoint.MediaFormat.md) object.


## Return value

Long
