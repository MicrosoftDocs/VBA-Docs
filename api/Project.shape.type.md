# Shape.Type Property (Project)
Gets the shape type. Read-only  **[MsoShapeType](https://msdn.microsoft.com/library/office/ff860759%28v=office.15%29)**.

## Syntax

 _expression_. `Type`

 _expression_ A variable that represents a 'Shape' object.


## Property value

 **MSOSHAPETYPE**
