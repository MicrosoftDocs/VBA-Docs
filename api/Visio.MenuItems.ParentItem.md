# MenuItems.ParentItem property (Visio)

Returns the parent object of a hierarchical menu. Read-only.


## Syntax

_expression_. `ParentItem`

_expression_ A variable that represents a **[MenuItems](Visio.MenuItems.md)** object.


## Return value

MenuItem
