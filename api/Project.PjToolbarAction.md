# PjToolbarAction enumeration (Project)

Contains constants that specify toolbar actions.



|Name|Value|Description|
|:-----|:-----|:-----|
|**pjCopy**|4|Copy.|
|**pjDelete**|2|Delete.|
|**pjHide**|7|Hide.|
|**pjNew**|3|New.|
|**pjRename**|5|Rename.|
|**pjReset**|2|Reset.|
|**pjShow**|6|Show.|
|**pjToggle**|1|Toggle.|
