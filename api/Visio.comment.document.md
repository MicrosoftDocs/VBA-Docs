# Comment.Document Property (Visio)

Returns the document that contains the comment. Read-only  ** **[Document](Visio.Document.md)****.


## Syntax

 _expression_. `Document`

 _expression_ A variable that represents a 'Comment' object.


## Property value

 **IVDOCUMENT**
