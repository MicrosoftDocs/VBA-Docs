# ShapeRange.Table property (Project)
Gets the **ReportTable** object in the shape range. Read-only[ReportTable](Project.reporttable.md).

## Syntax

_expression_. `Table`

_expression_ A variable that represents a 'ShapeRange' object.


## Property value

 **REPORTTABLE**
