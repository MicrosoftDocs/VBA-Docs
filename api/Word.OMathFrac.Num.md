# OMathFrac.Num property (Word)

Returns an  **[OMath](Word.OMath.md)** object that represents the numerator for a fraction. Read-only.


## Syntax

 _expression_. `Num`

 _expression_ An expression that returns an '[OMathFrac](Word.OMathFrac.md)' object.
