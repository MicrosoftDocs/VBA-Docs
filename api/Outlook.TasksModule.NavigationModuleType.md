# TasksModule.NavigationModuleType property (Outlook)

Returns an **[OlNavigationModuleType](Outlook.OlNavigationModuleType.md)** constant that represents the navigation module type for the object. Read-only.


## Syntax

_expression_. `NavigationModuleType`

 _expression_ An expression that returns a [TasksModule](Outlook.TasksModule.md) object.


## Remarks

This property returns  **olModuleTasks** for the **[TasksModule](Outlook.TasksModule.md)** object.
