# PjFieldType enumeration (Project)

Contains constants that specify the type of entity for a field.



|Name|Value|Description|
|:-----|:-----|:-----|
|**pjProject**|2|Project entity.|
|**pjResource**|1|Resource entity.|
|**pjTask**|0|Task entity.|
