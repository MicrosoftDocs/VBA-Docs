# DataLabels.Format property (Excel)

Returns the **[ChartFormat](Excel.ChartFormat.md)** object. Read-only.


## Syntax

_expression_.**Format**

_expression_ A variable that represents a **[DataLabels](Excel.DataLabels(object).md)** object.
