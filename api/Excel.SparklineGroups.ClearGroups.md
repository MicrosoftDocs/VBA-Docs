# SparklineGroups.ClearGroups method (Excel)

Clears the selected sparkline groups.


## Syntax

_expression_. `ClearGroups`

_expression_ A variable that represents a '[SparklineGroups](Excel.SparklineGroups.md)' object.


## Return value

Nothing
