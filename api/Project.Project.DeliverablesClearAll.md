# Project.DeliverablesClearAll Method (Project)

Clears all deliverables in the project.


## Syntax

 _expression_. `DeliverablesClearAll`

 _expression_ An expression that returns a [Project](./Project(enumerations).md) object.


## Return value

 **Boolean**
