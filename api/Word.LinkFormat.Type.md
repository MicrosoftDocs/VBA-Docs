# LinkFormat.Type property (Word)

Returns the link type. Read-only  **[WdLinkType](Word.WdLinkType.md)**.


## Syntax

_expression_.**Type**

_expression_ Required. A variable that represents a '[LinkFormat](Word.LinkFormat.md)' object.
