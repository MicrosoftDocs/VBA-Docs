# LegendKey.Delete method (PowerPoint)

Deletes the object.


## Syntax

_expression_.**Delete**

_expression_ A variable that represents a '[LegendKey](PowerPoint.LegendKey.md)' object.


## Remarks

Deleting a **LegendKey** object deletes the entire series.
