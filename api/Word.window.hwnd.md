# Window.Hwnd property (Word)

Returns a  **Long** that indicates the window handle of the specified window. Read-only.


## Syntax

_expression_. `Hwnd`

_expression_ A variable that represents a [Window](./Word.Window.md) object.


## Property value

 **INT32**
