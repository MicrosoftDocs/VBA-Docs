# NoteItem.Width property (Outlook)

Returns or sets a  **Long** indicating the width (in pixels) of the specified object. Read/write.


## Syntax

_expression_.**Width**

_expression_ A variable that represents a [NoteItem](Outlook.NoteItem.md) object.
