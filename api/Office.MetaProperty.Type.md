# MetaProperty.Type property (Office)

Gets the data type of a **MetaProperty** object. Read-only.


## Syntax

_expression_.**Type**

_expression_ An expression that returns a **[MetaProperty](Office.MetaProperty.md)** object.


## Return value

MsoMetaPropertyType
