# Workbook.Model property (Excel)

Returns the top level  **Model** object which is the one Data Model for the workbook. Read-only


## Syntax

_expression_. `Model`

_expression_ A variable that represents a [Workbook](./Excel.Workbook.md) object.


## Property value

 **MODEL**
