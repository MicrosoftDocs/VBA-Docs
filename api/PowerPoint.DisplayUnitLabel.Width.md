# DisplayUnitLabel.Width Property (PowerPoint)

Returns the width, in points, of the object. Read-only.


## Syntax

 _expression_. `Width`

 _expression_ A variable that represents a [DisplayUnitLabel](./PowerPoint.DisplayUnitLabel.md) object.


### Return value

Double
