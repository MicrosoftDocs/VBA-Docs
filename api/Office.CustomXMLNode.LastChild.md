# CustomXMLNode.LastChild property (Office)

Gets a **CustomXMLNode** object corresponding to the last child element of the current node. If the node has no child elements (or if it is not of type **[msoCustomXMLNodeElement](office.msocustomxmlnodetype.md)**), the property returns **Nothing**. Read-only.


## Syntax

_expression_.**LastChild**

_expression_ An expression that returns a **[CustomXMLNode](Office.CustomXMLNode.md)** object.


## Return value

CustomXMLNode
