# Label.Layout property (Access)

Returns the type of layout for the specified label. Read-only **[AcLayoutType](Access.AcLayoutType.md)**.


## Syntax

_expression_.**Layout**

_expression_ A variable that represents a **[Label](Access.Label.md)** object.
