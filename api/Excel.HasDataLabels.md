# HasDataLabels Property

 **True** if the series has data labels. Read/write **Boolean**.


## Example

This example turns on data labels for series three.


```vb
With myChart.SeriesCollection(3) 
 .HasDataLabels = True 
 .ApplyDataLabels Type:=xlValue 
End With
```
