# CustomLayout.Height property (PowerPoint)

Returns the height of the specified object, in points. Read-only.


## Syntax

_expression_.**Height**

_expression_ A variable that represents a [CustomLayout](PowerPoint.CustomLayout.md) object.


## Return value

Single
