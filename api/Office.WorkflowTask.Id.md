# WorkflowTask.Id property (Office)

Gets the ID of the Sharepoint list item. Read-only.


## Syntax

_expression_.**Id**

_expression_ An expression that returns a **[WorkflowTask](Office.WorkflowTask.md)** object.


## Return value

String
