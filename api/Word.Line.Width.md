# Line.Width property (Word)

Returns or sets the width, in [points](../language/glossary/vbe-glossary.md#point), of a line. Read/write  **Long**.


## Syntax

_expression_.**Width**

_expression_ A variable that represents a '[Line](Word.Line.md)' object.
