# PageSizes.Parent Property (Publisher)

Returns the parent of the  **[PageSizes](Publisher.PageSizes.md)** collection. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a  **PageSizes** collection.


## Return value

Object
