# ODSOColumn.Value property (Office)

Gets the value of a data field in a mail merge data source. Read-only.


## Syntax

_expression_. `Value`

_expression_ A variable that represents an [ODSOColumn](Office.ODSOColumn.md) object.


## Return value

String
