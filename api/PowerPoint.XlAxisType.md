# xlAxisType Enumeration (PowerPoint)

Specifies the axis type.



|Name|Value|Description|
|:-----|:-----|:-----|
|**xlCategory**|1|Axis displays categories.|
|**xlSeriesAxis**|3|Axis displays data series.|
|**xlValue**|2|Axis displays values.|
