# Timing.Duration Property (PowerPoint)

Returns or sets the length of an animation in seconds. Read/write.


## Syntax

 _expression_. `Duration`

 _expression_ A variable that represents a [Timing](./PowerPoint.Timing.md) object.


## Property value

 **FLOAT**
