# TableOfAuthorities.Update method (Word)

Updates the entries shown in a table of authorities.


## Syntax

 _expression_. `Update`

 _expression_ Required. A variable that represents a '[TableOfAuthorities](Word.TableOfAuthorities.md)' collection.
