# Point.Copy Method (Excel)

If the point has a picture fill, then this method copies the picture to the Clipboard.


## Syntax

 _expression_. `Copy`

 _expression_ A variable that represents a [Point](Excel.Point(Graph object).md) object.


### Return value

Variant
