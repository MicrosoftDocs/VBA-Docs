# OMathBreaks.Count property (Word)

Returns the number of items in the **OMathBreaks** collection. Read-only **Long**.


## Syntax

_expression_.**Count**

_expression_ A variable that represents a '[OMathBreaks](Word.OMathBreaks.md)' object.
