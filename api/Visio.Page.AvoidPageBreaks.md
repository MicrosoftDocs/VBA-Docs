# Page.AvoidPageBreaks Method (Visio)

Makes small adjustments to shapes to move them off page breaks.


## Syntax

 _expression_. `AvoidPageBreaks`

 _expression_ A variable that represents a '[Page](Visio.Page.md)' object.


### Return value

 **Nothing**
