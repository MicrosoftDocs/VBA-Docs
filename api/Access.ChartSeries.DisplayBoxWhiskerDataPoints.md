# ChartSeries.DisplayBoxWhiskerDataPoints property (Access)

**True** if data points are displayed for a Box and Whisker Modern Chart (only available in Current Channel). Read/write **Boolean**.


## Syntax

_expression_.**DisplayBoxWhiskerDataPoints**

_expression_ A variable that represents a **[ChartSeries](Access.ChartSeries.md)** object.
