# FormRegion.Item property (Outlook)

Returns an **Object** corresponding to the Outlook item bound to the form region. Read-only.


## Syntax

_expression_.**Item**

_expression_ A variable that represents a [FormRegion](Outlook.FormRegion.md) object.
