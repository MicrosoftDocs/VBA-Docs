# MasterShortcut.Prompt property (Visio)

Gets or sets the prompt string for a master or master shortcut. Read/write.


## Syntax

_expression_. `Prompt`

_expression_ A variable that represents a **[MasterShortcut](Visio.MasterShortcut.md)** object.


## Return value

String
