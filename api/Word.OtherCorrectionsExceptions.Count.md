# OtherCorrectionsExceptions.Count property (Word)

Returns a  **Long** that represents the number of exceptions in the collection. Read-only.


## Syntax

 _expression_. `Count`

 _expression_ Required. A variable that represents an '[OtherCorrectionsExceptions](Word.othercorrectionsexceptions.md)' collection.
