# DistListItem.Size property (Outlook)

Returns a  **Long** indicating the size (in bytes) of the Outlook item. Read-only.


## Syntax

_expression_.**Size**

_expression_ A variable that represents a [DistListItem](Outlook.DistListItem.md) object.
