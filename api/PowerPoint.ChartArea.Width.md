# ChartArea.Width Property (PowerPoint)

Returns or sets the width, in points, of the object. Read/write  **Double**.


## Syntax

 _expression_. `Width`

_expression_ A variable that represents a '[ChartArea](PowerPoint.ChartArea.md)' object.
