# xlLookFor enumeration (Excel)

Specifies what to look for in searches.



|Name|Value|Description|
|:-----|:-----|:-----|
| **LookForBlanks**|0|Blanks|
| **LookForErrors**|1|Errors|
| **LookForFormulas**|2|Formulas|
