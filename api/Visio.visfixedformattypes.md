# VisFixedFormatTypes Enumeration (Visio)

File format types passed to the  **Document.ExportAsFixedFormat** method.



|Name|Value|Description|
|:-----|:-----|:-----|
| **visFixedFormatPDF**|1|PDF format.|
| **visFixedFormatXPS**|2|XPS format.|
