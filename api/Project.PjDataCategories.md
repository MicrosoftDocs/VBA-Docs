# PjDataCategories enumeration (Project)

Contains constants that specify the category of data.



|Name|Value|Description|
|:-----|:-----|:-----|
|**pjMapAssignments**|2|Map assignments.|
|**pjMapResources**|1|Map resources.|
|**pjMapTasks**|0|Map tasks.|
