# PlotArea.Height Property (PowerPoint)

Returns or sets the height, in points, of the object. Read/write  **Double**.


## Syntax

 _expression_. `Height`

 _expression_ A variable that represents a '[PlotArea](PowerPoint.PlotArea.md)' object.
