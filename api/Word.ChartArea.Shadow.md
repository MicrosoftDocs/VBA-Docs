# ChartArea.Shadow property (Word)

Returns or sets a value that determines whether the object has a shadow. Read/write  **Boolean**.


## Syntax

_expression_.**Shadow**

_expression_ A variable that represents a '[ChartArea](Word.ChartArea.md)' object.
