# ShapeRange.Delete Method (PowerPoint)

Deletes the specified  **ShapeRange** object.


## Syntax

 _expression_. `Delete`

 _expression_ A variable that represents a [ShapeRange](./PowerPoint.ShapeRange.md) object.
