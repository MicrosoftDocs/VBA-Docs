# ProtectedViewWindow.Activate Method (PowerPoint)

Activates the specified window.


## Syntax

 _expression_. `Activate`

 _expression_ A variable that represents a [ProtectedViewWindow](./PowerPoint.ProtectedViewWindow.md) object.
