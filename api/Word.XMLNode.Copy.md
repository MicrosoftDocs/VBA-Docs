# XMLNode.Copy method (Word)

Copies the specified XML element, excluding XML markup, to the Clipboard.


## Syntax

_expression_.**Copy**

 _expression_ An expression that returns an [XMLNode](./Word.XMLNode.md) object.


## Return value

Nothing
