# ThemeFont.Name Property (Office)

Gets or sets the name of a font in the font scheme of a Microsoft Office theme. Read/write.


## Syntax

 _expression_. `Name`

 _expression_ An expression that returns a [ThemeFont](./Office.ThemeFont.md) object.


### Return Value

String
