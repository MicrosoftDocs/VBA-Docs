# LegendEntry.Font property (Word)

Returns the font of the specified object. Read-only  **[ChartFont](Word.ChartFont.md)**.


## Syntax

 _expression_. `Font`

 _expression_ A variable that represents a '[LegendEntry](Word.LegendEntry.md)' object.
