# TableStyle.Delete Method (Excel)

Deletes the  **[TableStyle](Excel.TableStyle.md)** object.


## Syntax

 _expression_. `Delete`

 _expression_ A variable that represents a [TableStyle](./Excel.TableStyle.md) object.


### Return value

Nothing
