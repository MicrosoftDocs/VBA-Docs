# AutoCorrect.Parent Property (Excel)

Returns the parent object for the specified object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ A variable that represents an [AutoCorrect](Excel.AutoCorrect(Graph property).md) object.
