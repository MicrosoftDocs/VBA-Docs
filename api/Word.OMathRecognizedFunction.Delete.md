# OMathRecognizedFunction.Delete method (Word)

Deletes the specified recognized function from the list of the recognized functions.


## Syntax

 _expression_. `Delete`

 _expression_ An expression that returns an [OMathRecognizedFunction](./Word.OMathRecognizedFunction.md) object.
