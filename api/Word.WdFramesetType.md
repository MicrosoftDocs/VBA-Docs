# WdFramesetType enumeration (Word)

Specifies the type of frame.



|Name|Value|Description|
|:-----|:-----|:-----|
| **wdFramesetTypeFrame**|1|A single frame.|
| **wdFramesetTypeFrameset**|0|A frameset.|
