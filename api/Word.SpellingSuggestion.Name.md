# SpellingSuggestion.Name property (Word)

Returns the name of the specified object. Read-only  **String**.


## Syntax

_expression_.**Name**

_expression_ Required. A variable that represents a '[SpellingSuggestion](Word.SpellingSuggestion.md)' object.
