# BorderArtFormat.Delete Method (Publisher)

Deletes the specified object.


## Syntax

 _expression_. **Delete**

 _expression_ A variable that represents a  **BorderArtFormat** object.


## Remarks

A run-time error occurs if the specified object does not exist.


## Example

The following example tests for the existence of BorderArt on each shape for each page of the active publication. If BorderArt exists, it is deleted.


```vb
Sub DeleteBorderArt() 
 
Dim anyPage As Page 
Dim anyShape As Shape 
 
For Each anyPage in ActiveDocument.Pages 
 For Each anyShape in anyPage.Shapes 
 With anyShape.BorderArt 
 If .Exists = True Then 
 .Delete 
 End If 
 End With 
 Next anyShape 
 Next anyPage 
End Sub
```
