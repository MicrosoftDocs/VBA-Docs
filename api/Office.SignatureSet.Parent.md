# SignatureSet.Parent Property (Office)

Gets the  **Parent** object for the **SignatureSet** object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ A variable that represents a [SignatureSet](./Office.SignatureSet.md) object.


### Return value

Object
