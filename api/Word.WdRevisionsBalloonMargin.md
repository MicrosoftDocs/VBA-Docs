# WdRevisionsBalloonMargin enumeration (Word)

Specifies the margin where revision balloons should be displayed.



|Name|Value|Description|
|:-----|:-----|:-----|
| **wdLeftMargin**|0|Left margin.|
| **wdRightMargin**|1|Right margin. default.|
