# PageSize.Parent Property (Publisher)

Returns the parent of the  **[PageSize](Publisher.PageSizes.md)** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a  **PageSize** object.


## Return value

Object
