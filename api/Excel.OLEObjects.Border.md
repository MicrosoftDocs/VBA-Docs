# OLEObjects.Border property (Excel)

Returns a  **[Border](Excel.Border(object).md)** object that represents the border of the object.


## Syntax

_expression_. `Border`

_expression_ A variable that represents an [OLEObjects](Excel.OLEObjects.md) object.
