# Columns.Parent property (Outlook)

Returns the parent object of the  **[Columns](Outlook.Columns.md)** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a [Columns](Outlook.Columns.md) object.
