# XmlNamespaces.Count property (Excel)

Returns a  **Long** value that represents the number of objects in the collection.


## Syntax

_expression_.**Count**

_expression_ A variable that represents a [XmlNamespaces](./Excel.XmlNamespaces.md) object.
