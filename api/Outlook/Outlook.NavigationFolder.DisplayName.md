# NavigationFolder.DisplayName Property (Outlook)

Returns a  **String** representing the display name of the **[NavigationFolder](Outlook.NavigationFolder.md)** object. Read-only.


## Syntax

 _expression_ . **DisplayName**

 _expression_ A variable that represents a **NavigationFolder** object.


## Remarks

Use  **NavigationFolder.DisplayName** , instead of **[Folder.Name](Outlook.Folder.Name.md)** , to get the name of a shared folder.
