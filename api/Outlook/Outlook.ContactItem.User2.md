# ContactItem.User2 Property (Outlook)

Returns or sets a  **String** specifying the second field on the Contacts form intended for miscellaneous use for the contact. Read/write.


## Syntax

 _expression_ . **User2**

 _expression_ A variable that represents a **ContactItem** object.


## Remarks

The properties  **[ContactItem.User1](Outlook.ContactItem.User1.md)** , **ContactItem.User2** , **[ContactItem.User3](Outlook.ContactItem.User3.md)** , and **[ContactItem.User4](Outlook.ContactItem.User4.md)** map to the fields **User Field 1**,  **User Field 2**,  **User Field 3**, and  **User Field 4** on the Contacts form respectively. These properties are explicit built-in **String** properties; users can use these fields for miscellaneous purposes for the contact.
