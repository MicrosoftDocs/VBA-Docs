# Shapes.Count property (Project)
Gets the number of  **Shape** objects in the **Shapes** collection. Read-only **Long**.

## Syntax

_expression_.**Count**

_expression_ A variable that represents a **[Shapes](Project.Shapes.md)** object.


## Property value

 **INT**
