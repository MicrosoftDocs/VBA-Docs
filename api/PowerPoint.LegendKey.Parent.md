# LegendKey.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. `Parent`

 _expression_ A variable that represents a '[LegendKey](PowerPoint.LegendKey.md)' object.
