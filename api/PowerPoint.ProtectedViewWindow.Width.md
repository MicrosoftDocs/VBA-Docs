# ProtectedViewWindow.Width property (PowerPoint)

Returns or sets the width of the specified object, in points. Read/write.


## Syntax

_expression_.**Width**

_expression_ A variable that represents a [ProtectedViewWindow](PowerPoint.ProtectedViewWindow.md) object.
