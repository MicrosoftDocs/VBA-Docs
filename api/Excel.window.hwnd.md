# Window.hWnd property (Excel)

Returns a  **Long** that indicates the window handle of the specified window. Read-only.


## Syntax

_expression_.**hWnd**

_expression_ A variable that represents a [Window](Excel.Window.md) object.


## Property value

 **INT32**
