# AutoFilter.Sort property (Excel)

Gets the sort column or columns, and sort order for the **AutoFilter** collection.


## Syntax

_expression_.**Sort**

_expression_ A variable that represents an **[AutoFilter](Excel.AutoFilter.md)** object.
