# Legend.Format Property (Excel)

Returns the  **[ChartFormat](Excel.ChartFormat.md)** object. Read-only.


## Syntax

 _expression_. `Format`

 _expression_ A variable that represents a [Legend](Excel.Legend(Graph property).md) object.
