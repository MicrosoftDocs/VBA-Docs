# WdAlignmentTabAlignment enumeration (Word)

Specifies tab alignment.



|Name|Value|Description|
|:-----|:-----|:-----|
| **wdCenter**|1|Centered tab.|
| **wdLeft**|0|Left-aligned tab.|
| **wdRight**|2|Right-aligned tab.|
