# CustomXMLPart.SchemaCollection property (Office)

Gets or sets a **[CustomXMLSchemaCollection](office.customxmlschemacollection.md)** object representing the set of schemas attached to a bound region of data in a document. Read/write.


## Syntax

_expression_.**SchemaCollection**

_expression_ An expression that returns a **[CustomXMLPart](Office.CustomXMLPart.md)** object.


## Return value

CustomXMLSchemaCollection
