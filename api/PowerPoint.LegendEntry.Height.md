# LegendEntry.Height Property (PowerPoint)

Returns the height, in points, of the object. Read-only  **Double**.


## Syntax

 _expression_. `Height`

 _expression_ A variable that represents a '[LegendEntry](PowerPoint.LegendEntry.md)' object.
