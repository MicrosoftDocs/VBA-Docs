# PivotFilters.Parent Property (Excel)

Returns the parent object for the specified  **PivotFilters** object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ A variable that represents a [PivotFilters](Excel.PivotFilters.md) object.
