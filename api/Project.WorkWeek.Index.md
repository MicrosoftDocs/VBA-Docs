# WorkWeek.Index property (Project)

Gets the index of a  **WorkWeek** object in the **WorkWeeks** containing object. Read-only **Long**.


## Syntax

_expression_.**Index**

 _expression_ An expression that returns a [WorkWeek](./Project.WorkWeek.md) object.


## Remarks

The  **Index** properties of different objects are used in similar ways. For an example, see the **[Index](Project.Project.Index.md)** property of the **Project** object.
