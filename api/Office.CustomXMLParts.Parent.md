# CustomXMLParts.Parent property (Office)

Gets the **Parent** object for the **CustomXMLParts** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ An expression that returns a **[CustomXMLParts](Office.CustomXMLParts.md)** object.


## Return value

Object
