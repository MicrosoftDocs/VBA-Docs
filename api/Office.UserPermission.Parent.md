# UserPermission.Parent property (Office)

Gets the **Parent** object for the **UserPermission** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a **[UserPermission](Office.UserPermission.md)** object.


## Return value

Object
