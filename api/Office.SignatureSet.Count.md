# SignatureSet.Count property (Office)

Gets a  **Long** indicating the number of items in the **SignatureSet** object. Read-only.


## Syntax

 _expression_. `Count`

 _expression_ A variable that represents a [SignatureSet](Office.SignatureSet.md) object.


## Return value

Long
