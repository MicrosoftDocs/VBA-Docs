# Assignment.BCWS Property (Project)

Gets the budgeted cost of work scheduled for the assignment. Read-only  **Variant**.


## Syntax

 _expression_. `BCWS`

 _expression_ A variable that represents an [Assignment](./Project.Assignment.md) object.
