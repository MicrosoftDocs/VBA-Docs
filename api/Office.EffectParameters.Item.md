# EffectParameters.Item property (Office)

Retrieves an **[EffectParameter](office.effectparameter.md)** object at the specified index or with the specified unique Id. Read-only.


## Syntax

_expression_.**Item**(_Index_)

_expression_ An expression that returns an **[EffectParameters](Office.EffectParameters.md)** object.


## Parameters

|Name|Required/Optional|Data type|Description|
|:-----|:-----|:-----|:-----|
| _Index_|Required|**Variant**|Specifies either an integer representing the index or a string representing the location of the **EffectParameter**.|
