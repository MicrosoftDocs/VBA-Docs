# Rule.Class property (Outlook)

Returns an **[OlObjectClass](Outlook.OlObjectClass.md)** constant indicating the object's class. Read-only.


## Syntax

_expression_.**Class**

_expression_ A variable that represents a [Rule](Outlook.Rule.md) object.
