# XlPriority enumeration (Excel)

Specifies the priority of a SendMailer message.



|Name|Value|Description|
|:-----|:-----|:-----|
| **xlPriorityHigh**|-4127|High|
| **xlPriorityLow**|-4134|Low|
| **xlPriorityNormal**|-4143|Normal|
