# Comment.CreateDate property (Visio)

Returns the date the specified comment was created. Read-only  **Date**.


## Syntax

_expression_.**CreateDate**

_expression_ A variable that represents a **[Comment](Visio.Comment.md)** object.


## Property value

 **DATE**
