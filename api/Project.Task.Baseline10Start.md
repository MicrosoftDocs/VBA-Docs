# Task.Baseline10Start Property (Project)

Gets or sets the baseline start date of a  **Task**. Read/write **Variant**.


## Syntax

 _expression_. `Baseline10Start`

 _expression_ A variable that represents a [Task](./Project.Task.md) object.
