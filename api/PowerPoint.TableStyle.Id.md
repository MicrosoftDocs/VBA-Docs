# TableStyle.Id Property (PowerPoint)

Returns a string that identifies the table style. Read-only.


## Syntax

 _expression_. `Id`

 _expression_ A variable that represents a [TableStyle](./PowerPoint.TableStyle.md) object.


## Return value

String
