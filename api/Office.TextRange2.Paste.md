# TextRange2.Paste method (Office)

Pastes the contents of the Clipboard into the **TextRange2** object.


## Syntax

_expression_.**Paste**

_expression_ An expression that returns a **[TextRange2](Office.TextRange2.md)** object.


## Return value

TextRange2
