# ODSOFilters.Parent Property (Office)

Gets the  **Parent** object for the **ODSOFilters** object. Read-only.


## Syntax

 _expression_. `Parent`

 _expression_ A variable that represents an [ODSOFilters](./Office.ODSOFilters.md) object.


### Return Value

Object
