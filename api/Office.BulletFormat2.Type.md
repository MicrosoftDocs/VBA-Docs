# BulletFormat2.Type Property (Office)

Gets or sets a constant that represents the type of bullet. Read/write.


## Syntax

 _expression_. `Type`

 _expression_ An expression that returns a [BulletFormat2](./Office.BulletFormat2.md) object.


### Return Value

MsoBulletType
