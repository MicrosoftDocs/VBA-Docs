# XlChartGallery enumeration (PowerPoint)

Specifies a chart gallery.



|Name|Value|Description|
|:-----|:-----|:-----|
|**xlAnyGallery**|23|Either of the galleries.|
|**xlBuiltIn**|21|The built-in gallery.|
|**xlUserDefined**|22|The user-defined gallery.|
