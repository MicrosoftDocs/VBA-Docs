# ThreeDFormat.BevelTopType property (PowerPoint)

Returns or set the top bevel type. Read/write.


## Syntax

_expression_. `BevelTopType`

_expression_ A variable that represents a [ThreeDFormat](PowerPoint.ThreeDFormat.md) object.


## Return value

MsoBevelType


## Remarks

The **BevelTopType** property value can be one of the **[MsoBevelType](Office.MsoBevelType.md)** constants.
