Contains constants that specify the task type.



|Name|Value|Description|
|:-----|:-----|:-----|
|**pjFixedDuration**|1|Fixed duration.|
|**pjFixedUnits**|0|Fixed units.|
|**pjFixedWork**|2|Fixed work.|
