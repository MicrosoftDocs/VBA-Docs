# LanguageSettings.Parent property (Office)

Gets the **Parent** object for the **ODSOFilter** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a **[LanguageSettings](Office.LanguageSettings.md)** object.


## Return value

Object
