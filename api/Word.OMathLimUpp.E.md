# OMathLimUpp.E property (Word)

Returns an **OMath** object that represents the base of the specified equation object. Read-only.


## Syntax

_expression_. `E`

 _expression_ An expression that returns an [OMathLimUpp](./Word.OMathLimUpp.md) object.
