# Values for the DefaultSource Member

**Applies to:** Access 2013 | Access 2016

|Value|Description|
|:-----|:-----|
|1|Upper or only one bin|
|2|Lower bin|
|3|Middle bin|
|4|Manual bin|
|5|Envelope bin|
|6|Envelope manual bin|
|7|Automatic bin|
|8|Tractor bin|
|9|Small-format bin|
|10|Large-format bin|
|11|Large-capacity bin|
|14|Cassette bin|
|256|Device-specific bins start here|
