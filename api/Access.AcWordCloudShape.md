# AcWordCloudShape enumeration (Access)

Specifies the shape for a Word Cloud Modern Chart.

|Name|Value|Description|
|:-----|:-----|:-----|
|**acWordCloudRectangle**|0|Rectangle|
|**acWordCloudOval**|1|Oval|
|**acWordCloudHeart**|2|Heart|
