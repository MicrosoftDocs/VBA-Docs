# ServerViewableItems.Count property (Excel)

Returns the number of objects in the collection. Read-only **Long**.


## Syntax

_expression_.**Count**

_expression_ A variable that represents a **[ServerViewableItems](Excel.ServerViewableItems.md)** object.
