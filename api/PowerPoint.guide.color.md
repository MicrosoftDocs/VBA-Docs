# Guide.Color property (PowerPoint)

Returns a [ColorFormat](PowerPoint.ColorFormat.md) object that represents the color of the drawing guide. Read-only.


## Syntax

_expression_.**Color**

_expression_ A variable that represents a 'Guide' object.


## Property value

 **COLORFORMAT**
