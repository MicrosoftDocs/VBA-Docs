# LookupTableEntry.Delete method (Project)

Deletes the **LookupTableEntry** object from a **LookupTable** collection.


## Syntax

_expression_.**Delete**

_expression_ A variable that represents a [LookupTableEntry](./Project.LookupTableEntry.md) object.
