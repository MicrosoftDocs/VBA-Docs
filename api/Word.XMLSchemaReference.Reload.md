# XMLSchemaReference.Reload method (Word)

Reloads the XML schemas that are referenced in a document.


## Syntax

 _expression_. `Reload`

 _expression_ An expression that returns an '[XMLSchemaReference](Word.XMLSchemaReference.md)' object.


## Return value

Nothing
