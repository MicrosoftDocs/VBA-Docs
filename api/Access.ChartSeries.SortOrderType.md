# ChartSeries.SortOrderType property (Access)

Returns or sets the sort order type for series in a Modern Chart (only available in Current Channel). Read/write **[AcSortOrderType](Access.AcSortOrderType.md)**.


## Syntax

_expression_.**SortOrderType**

_expression_ A variable that represents a **[ChartSeries](Access.ChartSeries.md)** object.
