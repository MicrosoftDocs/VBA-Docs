# ShadowFormat.Style Property (PowerPoint)

Returns or sets the shadow style. Read/write.


## Syntax

 _expression_. `Style`

 _expression_ A variable that represents a [ShadowFormat](./PowerPoint.ShadowFormat.md) object.


### Return value

MsoShadowStyle
