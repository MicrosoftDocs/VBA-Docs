# ChartSeries.DisplayBoxWhiskerMeanMarker property (Access)

**True** if the mean marker is displayed for a Box and Whisker Modern Chart (only available in Current Channel). Read/write **Boolean**.


## Syntax

_expression_.**DisplayBoxWhiskerMeanMarker**

_expression_ A variable that represents a **[ChartSeries](Access.ChartSeries.md)** object.
