# DataLabel.Format Property (Excel)

Returns the  **[ChartFormat](Excel.ChartFormat.md)** object. Read-only.


## Syntax

 _expression_. `Format`

 _expression_ A variable that represents a [DataLabel](Excel.DataLabel(Graph property).md) object.
