# AddressEntry.Address Property (Outlook)

Returns or sets a  **String** representing the e-mail address of the **[AddressEntry](Outlook.AddressEntry.md)** . Read/write.


## Syntax

 _expression_ . **Address**

 _expression_ A variable that represents an **AddressEntry** object.


## Remarks

The  **Address** property must be set before calling the **[AddressEntry.Details](Outlook.AddressEntry.Details.md)** method.
