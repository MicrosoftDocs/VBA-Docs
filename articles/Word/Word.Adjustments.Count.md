# Adjustments.Count Property (Word)

Returns the number of items in the  **Adjustments** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents an **[Adjustments](Word.Adjustments.md)** collection.
