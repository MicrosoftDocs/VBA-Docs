# EmptyCell.Layout Property (Access)

Returns the type of layout for the specified text box. Read-only  **[AcLayoutType](Access.AcLayoutType.md)**.


## Syntax

 _expression_. **Layout**

 _expression_ A variable that represents an **EmptyCell** object.
