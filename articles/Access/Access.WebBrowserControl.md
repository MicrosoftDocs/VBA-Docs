# WebBrowserControl Object (Access)

Represents a Web browser control on a form.


## Remarks

Use the [ControlSource](http://msdn.microsoft.com/library/f15e6d9c-fa41-8ca9-5252-fbb86139b2dc%28Office.15%29.aspx) property to specify the Web page to load in the control.
