# ComboBox.Layout Property (Access)

Returns the type of layout for the specified combo box. Read-only  **[AcLayoutType](Access.AcLayoutType.md)**.


## Syntax

 _expression_. **Layout**

 _expression_ A variable that represents a **ComboBox** object.
