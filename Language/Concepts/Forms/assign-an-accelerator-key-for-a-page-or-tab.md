# Assign an accelerator key for a Page or Tab

1. In design mode, select an individual **[Page](../../reference/user-interface-help/page-object.md)** or **[Tab](../../reference/user-interface-help/tab-object.md)**. Be sure to select the **Page** or **Tab**, not the associated **[MultiPage](../../reference/user-interface-help/multipage-control.md)** or **[TabStrip](../../reference/user-interface-help/tabstrip-control.md)**. When a **Page** or **Tab** is selected, a rectangle appears around the caption of the **Page** or **Tab**.
    
2. Right-click the selected **Page** or **Tab**.
    
3. From the shortcut menu, choose **Rename**.
    
4. In the **[Rename](../../reference/user-interface-help/the-rename-dialog-box-microsoft-forms.md)** dialog box, enter a single character in the **Accelerator Key** field.
    
> [!TIP] 
> Use a character from the caption of the control. Note that the selected character is underlined in the control's caption.
