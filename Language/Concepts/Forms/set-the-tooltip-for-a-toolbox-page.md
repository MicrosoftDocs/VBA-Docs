# Set the ToolTip for a Toolbox page

1. Select the page of the **[Toolbox](../../reference/user-interface-help/toolbox.md)**.
    
2. Right-click.
    
3. From the shortcut menu, choose **Rename**.
    
4. Enter the new text for the ToolTip.
    
5. Choose **OK**.
