# Export Method (VBA Add-In Object Model)



Saves a component as a separate file or files.

## Syntax

_object_**.Export(**_filename_**)**
The  **Export** syntax has these parts:


|**Part**|**Description**|
|:-----|:-----|
| _object_|Required. An [object expression](../../Glossary/vbe-glossary.md#object-expression) that evaluates to an object in the Applies To list.|
| _filename_|Required. A [String](../../Glossary/vbe-glossary.md#String) specifying the name of the file that you want to export the component to.|
