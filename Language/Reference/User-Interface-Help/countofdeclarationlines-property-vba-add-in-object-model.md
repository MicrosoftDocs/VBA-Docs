# CountOfDeclarationLines Property (VBA Add-In Object Model)



Returns a [Long](../../Glossary/vbe-glossary.md#Long) containing the number of lines of code in the Declarations section of a[code module](../../Glossary/vbe-glossary.md#code-module). Read-only.
