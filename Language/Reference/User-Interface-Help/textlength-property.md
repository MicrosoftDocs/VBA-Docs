# TextLength property

Returns the length, in characters, of text in the edit region of a **[TextBox](textbox-control.md)** or **[ComboBox](combobox-control.md)**.

## Syntax

_object_.**TextLength**

The **TextLength** property syntax has these parts:

|Part|Description|
|:-----|:-----|
| _object_|Required. A valid object.|

## Remarks

The **TextLength** property is read-only. 

For a multiline **TextBox**, **TextLength** includes LF (line feed) and CR (carriage return) characters.
