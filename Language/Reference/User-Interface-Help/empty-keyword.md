# Empty <keyword>

The  **Empty**[keyword](../../Glossary/vbe-glossary.md#keyword) is used as a[Variant](../../Glossary/vbe-glossary.md#Variant) subtype. It indicates an uninitialized[variable](../../Glossary/vbe-glossary.md#variable) value.
