# ODBC Specification Conformance Error \<message\>. Report this error to the developer of your application. (Error 3299)

  

**Applies to:** Access 2013 | Access 2016

This error can be caused by any of a number of problems with your ODBC driver. Write down the error number indicated in the message and report it to the developer of the application, who can then report it to the driver vendor.
