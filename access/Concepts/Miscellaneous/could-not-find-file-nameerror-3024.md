# Could not find file <name>. (Error 3024)

  

**Applies to:** Access 2013 | Access 2016

Possible causes:



- The specified file does not exist.
    
- You misspelled the file name. Check for missing underscores ( _ ) or other punctuation, and make sure you did not enter leading spaces.
    
- The network connection has failed or has not been established.
