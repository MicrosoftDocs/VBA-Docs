# The size of a field is too long. (Error 3017)

  

**Applies to:** Access 2013 | Access 2016

Possible causes:



- You tried to define or create a Text field longer than 254 characters. Use a number between 1 and 254, and then try the operation again, or create a Memo field for the data.
    
- You tried to define a query parameter longer than 255 characters.
